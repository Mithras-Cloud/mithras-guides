<h3>Desconectar tu web</h3>

Tu sitio web creado con PXSOL se desconectará automáticamente una vez que tu baja del sistema haya sido procesada.

**A - Si tu dominio fue comprado por PXSOL:**

Puedes solicitar la transferencia del dominio a tu nombre. Esto te permitirá desarrollar un nuevo sitio web fuera de PXSOL utilizando el mismo dominio.

<a
  href="https://ayuda.pxsol.com/es/solicitar-transferencia-de-dominio-comprado-por-pxsol"
  target="_blank"
  rel="noopener"
>
  Solicitar transferencia de dominio comprado por PXSOL
</a>

<Note>
  PXSOL solo podrá transferir el dominio a tu nombre hasta la próxima fecha de vencimiento. Una vez
  superada esa fecha, el dominio no será renovado por nuestra parte.
</Note>

**B - Si el dominio ya está a tu nombre:**

Debes acceder a tu registrador de dominios y modificar los servidores de nombre (NS). Reemplaza los servidores de Cloudflare actuales por los de tu proveedor de hosting web. Este proceso es similar al que realizaste al dar de alta tu web.

<Note>
  Si has dado de baja tu hosting web anterior, deberás contratar un nuevo servicio de hosting para
  poder alojar tu página. Si mantuviste tu hosting anterior, puedes reconfigurar tus NS previos y tu
  antigua web se activará automáticamente.
</Note>
