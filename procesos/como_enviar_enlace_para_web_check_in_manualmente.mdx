---
title: "¿Cómo enviar enlace para Web Check In manualmente?"
description: "En esta guía explicaremos cómo enviar al cliente el enlace para que realice el proceso de Check In online, a través de un correo o a través de Whatsapp."
---


<Steps>
<Step title="Ingresar a la reserva">
El enlace se podrá enviar de dos formas:
<ul>
<li>A través de correo electrónico</li>
<li>A través de Whatsapp</li>
</ul>
</Step>

<Step title="Enviar por Correo Electrónico">
Para enviar el enlace por correo electrónico, sigue estos pasos:
<ol>
<li>Selecciona el ícono de <strong>Responder</strong>. Se mostrará una imagen del ícono.</li>
<li>Selecciona la plantilla diseñada para enviar el enlace. Esta plantilla contendrá el link al cual el huésped podrá ingresar para cargar sus datos. Se mostrarán imágenes de ejemplo de la plantilla y los campos a completar.</li>
<li>Selecciona <strong>Enviar</strong>. Se mostrará una imagen del botón de enviar.</li>
</ol>
</Step>

<Step title="Enviar por Whatsapp">
Para enviar el enlace por Whatsapp, sigue estos pasos:
<ol>
<li>Una vez ingresado al ícono de Whatsapp, selecciona <strong>Compartir Link Web Check In</strong>. Se mostrará una imagen del ícono a seleccionar.</li>
<li>Se abrirá una nueva ventana de Whatsapp Web para que puedas enviarlo al cliente correspondiente.</li>
</ol>
</Step>
</Steps>
