title: "Gestión de Reservas"
description: "En esta guía veremos cómo utilizar las aplicaciones Cargar Reservas y Ocupación."
---


A partir del plan All in One se suman a nuestro sistema aplicaciones como Cargar Reservas y Ocupación. Estas aplicaciones nos permitirán registrar una Reserva en la instancia Preventa o Venta y visualizarla en un Plano de ocupación que lo mostrará de una forma ordenada y de fácil interpretación.

Recomendamos leer las guías de planes anteriores, para luego, ver las aplicaciones que agrega cada Plan.

En este caso sugerimos leer la siguiente guía que explica todos los posibles canales por donde puede ingresar una reserva y cómo gestionar cada uno:

[Proceso 3 - Gestión de Consultas](https://ayuda.pxsol.com/ciclo-de-vida-de-una-reserva-plan-free)

Existe la posibilidad de que el cliente esté decidido en reservar una habitación para cierta fecha y que haya confirmado el método de pago. En ese caso se podrá registrar esta consulta como una Reserva en la instancia Venta. La cual puede ingresar por los siguientes canales:

<Frame>
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-Jun-29-2020-08-52-38-15-PM.png" alt="Canales de ingreso de reservas" />
</Frame>

En este Plan se suma la posibilidad de registrar una Preventa o una Venta a partir de la aplicación Cargar Reservas. Para utilizarla deberá seguir los pasos de la siguiente guía:

[¿Cómo cargar una reserva?](https://ayuda.pxsol.com/como-cargar-una-reserva)

Una vez cargada la reserva podremos ingresar en el Plano de Ocupación, lo que permitirá visualizar claramente, las reservas en la grilla de ocupación y poder realizar movimientos de habitación. Para ingresar al Plano y entender su funcionamiento recomendamos visualizar la siguiente guía:

[¿Cómo visualizar el plano de ocupación?](https://ayuda.pxsol.com/como-visualizar-el-plano-de-ocupacion)

Una vez que ingresa al Plano de Ocupación podrá realizar configuraciones como:

*   Modificar Fechas de estadía
*   Cambiar Asignación de Reservas
*   Realizar bloqueos
*   Configurar colores

Todas estas acciones y algunas recomendaciones estarán explicadas en las guías que se encuentran en el siguiente link:

[Guías: Plano de Ocupación](https://ayuda.pxsol.com/pms#plano-de-ocupaci%C3%B3n)

Este Plan permite también, llevar un control del estado de limpieza de todas las habitaciones con las que cuenta nuestro hotel, para conocer si está libre e inspeccionada para la venta y para que las mucamas estén prevenidas de los ingresos, egresos y cambio de sábanas.

Para entender el funcionamiento de esta aplicación recomendamos leer la siguiente guía:

[¿Cómo leer Estado de habitaciones?](https://ayuda.pxsol.com/como-leer-estado-de-habitaciones)

Para configurar esta aplicación y acceder a los reportes vinculados deberá leer las guías que se encuentran en el siguiente link:

[Guías: Estado de habitaciones](https://ayuda.pxsol.com/pms#estado-de-habitaciones)
