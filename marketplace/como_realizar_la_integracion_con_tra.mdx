# ¿Cómo realizar la integración con TRA?

En esta guía explicaremos cómo realizar la integración con Tarjeta de Registro de Alojamiento (TRA) para automatizar el envío de los datos de los huéspedes a este sistema.

<Steps>
<Step title="Navegar a la sección de Integraciones">
  Ingresa desde la barra lateral a **Configuración - Integraciones**.
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-04-2025-12-18-25-7535-PM.png" alt="Captura de pantalla de la sección Configuración - Integraciones" />
  </Frame>
</Step>

<Step title="Instalar la integración TRA">
  Sobre la Tarjeta de TRA, selecciona **Instalar** y luego confirma con **Sí, Instalar**.
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-28-2024-02-52-25-4326-PM.png" alt="Captura de pantalla del botón Instalar TRA" />
  </Frame>
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-28-2024-02-53-23-8456-PM.png" alt="Confirmación de instalación de TRA" />
  </Frame>
</Step>

<Step title="Agregar Credencial">
  Selecciona el botón **Agregar Credencial**.
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-28-2024-02-54-06-6593-PM.png" alt="Botón Agregar Credencial" />
  </Frame>
</Step>

<Step title="Completar el formulario de credenciales">
  Completa todos los datos solicitados en la ventana emergente.
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-28-2024-02-54-52-1362-PM.png" alt="Formulario de credenciales de TRA" />
  </Frame>
  
  <Info>
  A continuación, se detalla la información necesaria:
  
  *   **Registro Nacional de Turismo (RNT):** Puedes obtener esta información desde <a href="https://rnt.confecamaras.co/home" target="_blank" rel="noopener">https://rnt.confecamaras.co/home</a>.
  *   **Token:** Este token lo puedes obtener desde <a href="https://pms.mincit.gov.co/token/" target="_blank" rel="noopener">https://pms.mincit.gov.co/token/</a>.
  *   **Envío Automático (SI/NO):**
      Si activas el envío automático, este se realizará en las siguientes instancias:
      *   Movimiento de la reserva a la instancia de *Check-in*.
      *   Instancia de *Check-out* (siempre que no se haya enviado manualmente antes).
      <Warning>
      Asegúrate de que los datos de los huéspedes sean correctos para evitar envíos erróneos. Si tienes dudas, solicita el envío manual. Para activar el envío automático, contacta al área de Soporte de Pxsol (equipo@pxsol.com).
      </Warning>
  *   **Establecimiento vinculado:** Selecciona el Establecimiento de tu cuenta en Pxsol que corresponde a esta vinculación. Aquí figurarán todos los establecimientos asociados a tu cuenta.
  </Info>
</Step>

<Step title="Guardar la configuración">
  Haz clic en el botón **Guardar** para finalizar la configuración.
  
  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-28-2024-02-57-07-4054-PM.png" alt="Botón Guardar" />
  </Frame>
  
  <Check>
  ¡La integración con TRA ha sido configurada exitosamente!
  </Check>
</Step>
</Steps>
