---
title: "¿Cómo generar un PIN para un Usuario?"
description: "En esta guía explicaremos cómo generar un PIN para nuevos usuarios."
---


<Note>
Solo un usuario con permiso de <strong>administrador</strong> puede otorgar PIN a usuarios que no posean este permiso. El mismo es necesario para poder realizar firmas. Esta firma virtual les permitirá anular y corroborar pagos y consumos.
Al ingresar el PIN para realizar estas acciones se vinculará el número de PIN con el usuario y quedará registro de la persona que realizó esa acción.
</Note>

<Steps>
<Step title="Acceder a la sección de Usuarios">
  Entra a la aplicación <strong>Usuarios</strong> desde el Omnibox.

  <Frame>
  <img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-05-2022-03-32-52-63-PM.png?width=479&amp;name=image-png-Sep-05-2022-03-32-52-63-PM.png" alt="Acceso a la aplicación Usuarios" />
  </Frame>
</Step>

<Step title="Seleccionar al usuario y Editar">
  En el listado de usuarios que aparece, haz clic en el <strong>menú de acciones</strong> a la derecha del usuario que deseas editar y selecciona la opción <strong>Editar</strong>.

  <Frame>
  <img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-05-2022-03-34-38-70-PM.png?width=330&amp;name=image-png-Sep-05-2022-03-34-38-70-PM.png" alt="Menú de acciones y opción Editar" />
  </Frame>
</Step>

<Step title="Habilitar Acceso Seguro">
  En la ventana que se despliega, baja hasta la sección <strong>Habilitar Acceso Seguro</strong> y activa el switch para habilitarlo.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-06-2022-01-11-50-10-PM.png" alt="Habilitar Acceso Seguro" />
  </Frame>
</Step>

<Step title="Guardar cambios iniciales">
  Una vez activado el switch, haz clic en <strong>Guardar cambios</strong> para poder continuar con la configuración del PIN.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-06-2022-01-13-40-76-PM.png" alt="Guardar cambios" />
  </Frame>
</Step>

<Step title="Configurar el PIN">
  Después de guardar los cambios, vuelve a entrar a la sección de usuarios y edita nuevamente el usuario. Ahora encontrarás una sección para configurar el PIN debajo de "Habilitar acceso seguro". Introduce un PIN de 4 números y vuelve a <strong>Guardar cambios</strong>.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-06-2022-01-16-53-02-PM.png" alt="Sección para configurar PIN" />
  </Frame>

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-06-2022-01-18-17-85-PM.png" alt="Introducir PIN de 4 números" />
  </Frame>

  <Note>
  Esta última acción, la de colocar el PIN, puede ser realizada por el mismo usuario al que se le ha habilitado el Acceso Seguro.
  </Note>
</Step>
</Steps>
