---
title: "¿Cómo generar un link de Motor de Reservas con código promocional ya cargado?"
description: "En esta guía veremos cómo editar el link del Motor de Reservas para que tenga cargado el Código Promocional al abrirlo."
---

Paso 1: Iniciar sesión en pms.pxsol.com con tus credenciales.

Paso 2: Accede a la aplicación "Motor de Reservas".

</Frame>

Paso 3: Modifica el enlace del Motor de Reservas. Añade `&promo_code=` al final del enlace, seguido del código promocional configurado. Por ejemplo, si tu código promocional es "PROMOINVIERNO", el enlace se verá así: `&promo_code=PROMOINVIERNO`.

Un ejemplo de enlace del Motor de Reservas con un código promocional incluido es:
`https://hotelescuela.reservadirecto.com/lp.html?pos=HotelEscuela&lng=es&cur=ARS&webcache=off&promo_code=PROMOINVIERNO`

Paso 4: Utiliza este enlace modificado para redireccionar ventanas emergentes (PopUps), campañas promocionales o para compartirlo con tus clientes especiales.
