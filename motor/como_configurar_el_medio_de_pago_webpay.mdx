---
title: "¿Cómo configurar WebPay como método de pago?"
description: "Esta guía detalla la configuración de WebPay, una de las principales plataformas de transacciones en línea en Chile, y cómo integrarla en tu motor de reservas para facilitar pagos rápidos y sencillos."
---

Para habilitar WebPay como método de pago en tu motor de reservas, necesitas tener al menos una credencial de conexión previamente configurada.

Si aún no has configurado tus credenciales, consulta la siguiente guía:

<Info>
  <a
    href="https://ayuda.pxsol.com/es/como-configurar-la-integracion-con-webpay"
    target="_blank"
    rel="noopener"
  >
    ¿Cómo configurar la integración con WebPay Plus?
  </a>
</Info>

Si ya dispones de una credencial de WebPay configurada, puedes proceder con los siguientes pasos para habilitar este método de pago en tu motor de reservas:

### 1. Acceder a la aplicación "Medios de pago"

Puedes encontrar la aplicación "Medios de pago" buscándola desde el **omnibox**:

<Frame>
  <img
    src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-17-2024-07-43-05-4319-PM.png?width=537&height=217&name=image-png-Sep-17-2024-07-43-05-4319-PM.png"
    alt="Búsqueda de la aplicación Medios de pago en el omnibox"
  />
</Frame>

### 2. Añadir un nuevo medio de pago

Al ingresar a la aplicación, selecciona el botón "**+ Nuevo medio de pago**" ubicado en la esquina superior derecha. Si deseas modificar un medio de pago existente, puedes hacerlo desde el menú de acciones a la derecha del medio de pago que desees editar.

<Frame>
  <img
    src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-17-2024-07-45-32-5620-PM.png?width=1706&height=279&name=image-png-Sep-17-2024-07-45-32-5620-PM.png"
    alt="Botón para añadir nuevo medio de pago o editar uno existente"
  />
</Frame>

### 3. Configurar la información del medio de pago

A continuación, se abrirá un panel lateral donde deberás ingresar la información requerida para la configuración.

#### 3.1. Categoría de medio de pago

En este primer paso, configura los siguientes aspectos:

- **Categoría**: Despliega el listado de alternativas disponibles y selecciona la opción **WebPay**.

<Note>
  La categoría WebPay solo estará disponible si cuentas con al menos una credencial configurada. De
  lo contrario, esta opción no aparecerá en el listado.
</Note>

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-07-56-40-4222-PM.png"
    alt="Selección de la categoría WebPay"
  />
</Frame>

- **Nombre único**: Este es el nombre con el que el huésped verá este medio de pago durante el proceso de reserva en el motor.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-08-00-12-5078-PM.png"
    alt="Ingreso del nombre único del medio de pago"
  />
</Frame>

#### 3.2. Configuración de cuenta recaudadora

Deberás elegir la cuenta recaudadora donde se procesarán los pagos realizados a través de esta pasarela. Solo se mostrarán las cuentas recaudadoras configuradas en la moneda CLP (pesos chilenos). Si aún no tienes una, puedes seleccionar "**Crear nueva cuenta recaudadora automáticamente**" para generarla instantáneamente.

<Note>
  Es fundamental configurar una cuenta recaudadora para garantizar que los pagos se reflejen
  correctamente en las reservas.
</Note>

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-08-04-44-0363-PM.png"
    alt="Selección o creación de cuenta recaudadora"
  />
</Frame>

#### 3.3. Configuración de credencial

Si tienes más de una credencial de WebPay configurada, podrás vincular este nuevo medio de pago a la credencial correspondiente.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-08-08-57-3614-PM.png"
    alt="Selección de credencial de WebPay"
  />
</Frame>

#### 3.4 / 3.5 / 3.6. Configuración de plantilla de correo automático

Aquí puedes asociar las plantillas de correo electrónico al método de pago. Podrás configurar el envío de un correo para la pre-confirmación de la reserva, así como un segundo correo que diferencie entre un pago exitoso y uno rechazado o con errores. Tienes la opción de seleccionar una plantilla existente o crear una nueva desde esta sección. ([Guía: ¿Cómo configurar una plantilla e-commerce?](https://ayuda.pxsol.com/es/como-configurar-las-plantillas-de-correo-e-commerce))

Te recomendamos diseñar plantillas específicas para cada estado del pago, lo que te ayudará a mantener una comunicación clara y efectiva con tus huéspedes.

Puedes revisar ejemplos de plantillas de correo electrónico para los distintos estados de la transacción en este [enlace](https://ayuda.pxsol.com/es/webpay-ejemplos-de-plantillas-de-correo-segun-el-estado-del-pago).

    alt="Configuración de plantillas de correo automático"

</Frame>

  <img

  />
</Frame>

<Warning>
  Sugerimos personalizar cada plantilla para cada idioma disponible en tu motor de reservas.
</Warning>

#### 3.6. Política asociada

En este apartado se mostrarán las políticas ya configuradas para tu establecimiento. Deberás seleccionar una o más de estas políticas para que apliquen a este medio de pago.

<Frame>
  <img
    src="https://ayuda.pxsol.com/hubfs/image-png-Sep-19-2022-05-28-02-62-PM.png"
    alt="Asociación de políticas al medio de pago"
  />
</Frame>

#### 3.7. Definir la moneda para el medio de pago

<Note>El medio de pago Webpay solo acepta la moneda CLP (pesos chilenos).</Note>

Selecciona la moneda CLP (pesos chilenos) para este medio de pago. WebPay solo opera con esta moneda. Si no seleccionas CLP, el sistema no permitirá avanzar en la configuración.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-19-41-3643-PM.png"
    alt="Selección de moneda CLP para WebPay"
  />
</Frame>

#### 3.8. Establecer si este medio de pago aplicará algún tipo de descuento o recargo

- **Recargo porcentual**: Puedes establecer un incremento porcentual que se aplicará al elegir este medio de pago.
- **Descuento porcentual**: Puedes definir un descuento porcentual que se aplicará al seleccionar esta opción de pago.
- **Cobrar solo un porcentaje**: Si deseas que el pago corresponda a un porcentaje específico del total.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-25-41-0184-PM.png"
    alt="Configuración de recargos o descuentos para el medio de pago"
  />
</Frame>

#### 3.9. Ventana de reserva

Establece el intervalo de días entre la fecha en que se realiza la compra y la fecha de llegada del huésped al hotel.

<Note>
  Por ejemplo, si deseas evitar reservas de último momento con este medio de pago, puedes establecer
  una **ventana de reserva mínima** de 1 o 2 días.
</Note>

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-28-06-2067-PM.png"
    alt="Configuración de la ventana de reserva"
  />
</Frame>

#### 3.10. Definir el punto de venta

Si cuentas con un único punto de venta, el medio de pago se aplicará automáticamente. Si tienes varios puntos de venta configurados, podrás seleccionar uno o más de ellos para que este medio de pago esté disponible.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-31-01-5390-PM.png"
    alt="Selección de punto de venta para el medio de pago"
  />
</Frame>

### 4. Guardar cambios

Finalmente, haz clic en el botón "**Guardar cambios**" para completar la configuración del medio de pago.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-33-48-1056-PM.png"
    alt="Botón Guardar cambios"
  />
</Frame>

Una vez configurado el medio de pago, en las búsquedas de tarifas dentro del motor de reservas, podrás visualizar los medios de pago que has creado:

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-19-2025-09-40-35-9614-PM.png"
    alt="Visualización de medios de pago en búsquedas de tarifas"
  />
</Frame>
