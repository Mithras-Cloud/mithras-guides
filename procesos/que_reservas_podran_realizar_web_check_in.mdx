---
title: "¿Qué reservas podrán realizar Web Check In?"
description: "En esta guía explicaremos todas las fuentes de ingreso de las reservas que podrán realizar el proceso de Web Check In."
---


Tendremos 3 fuentes de información por las que ingresan las reservas:

*   **Canales de Ventas conectados a Pxsol:** Incluye:
    *   Motor de Reservas del Hotel.
    *   Channel Manager: recibe reservas realizadas a través de Agencias de Viaje.
    *   CRM: recibe reservas realizadas a través de envío de correo electrónico.
*   **Carga masiva de Reservas externas:** Mediante la aplicación Importar Reservas, que permite cargar archivos en formato CSV, podrá cargar reservas externas a la plataforma de Pxsol, que pueden ingresar por:
    *   Agencias Tradicionales Offline.
    *   Canales de venta no vinculados al sistema.
    *   Ventas de mostrador, etc.
*   **Carga individual de Reservas:** Mediante la aplicación Cargar Reservas que permite cargar reservas individuales.

Estas fuentes por las que ingresan las reservas permitirán utilizar el proceso de Web Check in para cualquier plan que haya contratado dentro de Pxsol. La diferencia radica en que mientras más completo sea el plan, o sea, tenga más cantidad de canales comerciales conectados (Motor de reservas, CRM, Channel Manager), menor será el trabajo adicional que deba realizar la propiedad para permitirle a sus huéspedes llevar a cabo el proceso de Web Check in.

*   Al tener todos los canales de venta vinculados a Pxsol, la mayoría de las reservas van a estar pasando por el sistema y así ya se conocerá:
    *   Fecha de Ingreso.
    *   Apellido.
    *   Nº de Reserva externo.
*   Para los clientes de Pxsol que tengan planes más pequeños, que tengan menor comercialización a través de Pxsol, van a poder importar de manera masiva o individual las reservas de canales externos. Para realizar este procedimiento deberá seguir los pasos de la siguiente guía:
    *   [¿Cómo importar reservas al sistema?](https://ayuda.pxsol.com/como-importar-reservas)
*   Para los clientes de Pxsol que cuenten con la aplicación Cargar Reserva podrán ingresar al sistema cualquier Reserva que haya ingresado al hotel ya sea por un canal Interno o Externo, siguiendo los pasos de las siguientes guías, según corresponda a si cuentan o no con el Plano de Ocupación:
    *   [¿Cómo cargar una reserva?](https://ayuda.pxsol.com/como-cargar-una-reserva)
    *   [¿Cómo cargar una reserva al no contar con Plano de Ocupación?](https://ayuda.pxsol.com/es/c%C3%B3mo-cargar-una-reserva-sin-plano-de-ocupaci%C3%B3n)
