---
title: "¿Cómo ajustar el ancho de las columnas?"
description: "En esta guía explicaremos las distintas formas de ajustar los anchos de las columnas de cualquier informe."
---


### Ajuste manual del ancho de columna

Este método te permite ajustar el ancho de cada columna de forma individual.

<Steps>
<Step title="Navega a Informes">
  Haz clic en la opción "Informes" en el menú principal.
  <Frame>
    <img src="https://ayuda.pxsol.com/hubfs/image-55.png" alt="Selección de la opción Informes" />
  </Frame>
</Step>

<Step title="Visualiza los reportes disponibles">
  Se mostrará una lista de todos los reportes disponibles.
  <Frame>
    <img src="https://ayuda.pxsol.com/hubfs/image-56.png" alt="Listado de reportes disponibles" />
  </Frame>
</Step>

<Step title="Abre un informe">
  Al hacer clic en un informe, se abrirá una nueva pestaña en tu navegador con el formato del reporte.
  <Frame>
    <img src="https://ayuda.pxsol.com/hubfs/image-585.png" alt="Formato de un informe abierto" />
  </Frame>
</Step>

<Step title="Ajusta el ancho de una columna">
  1. Acerca el cursor del mouse al borde donde se unen los títulos de dos columnas.
  2. Cuando el cursor cambie al símbolo de flecha doble horizontal (↔), haz clic y arrastra el borde hasta alcanzar el ancho deseado para la columna.
</Step>

<Step title="Ajusta al ancho mínimo automáticamente">
  Para ajustar rápidamente el ancho de una columna al mínimo necesario para mostrar todo el texto, haz doble clic en el borde que separa los títulos de las columnas.
</Step>
</Steps>

### Ajuste automático de ancho para todas las columnas

Este método ajusta el ancho de todas las columnas de la tabla de manera uniforme.

<Steps>
<Step title="Ubica el cursor en el título de una columna">
  Acerca el cursor del mouse al título de cualquier columna. Verás aparecer un símbolo específico.
  <Frame>
    <img src="https://ayuda.pxsol.com/hs-fs/hubfs/Screenshot%202020-06-10%20at%2011-20-45.png?width=62&amp;name=Screenshot%202020-06-10%20at%2011-20-45.png" alt="Símbolo para ajustar columnas" width="62" />
  </Frame>
</Step>

<Step title="Selecciona la opción de ajuste automático">
  Se desplegará un menú. Selecciona la opción "Ajustar ancho automáticamente para toda la tabla".
  <Frame>
    <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-626.png" alt="Menú de configuración de columnas" />
  </Frame>
  <Check>
  Todas las columnas de tu informe se ajustarán automáticamente para una visualización óptima.
  </Check>
</Step>
</Steps>
