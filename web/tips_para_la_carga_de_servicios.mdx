---
title: "Tips para la carga de servicios"
description: "En esta guía les mostraremos algunos tips para la configuración de los servicios con los que cuenta el hotel."
---

Para cargar los servicios que aparecen en la Web van a utilizar una aplicación de Pxsol que les va a ofrecer un listado muy largo de servicios para que seleccionen.

<Frame>
  <img
    src="https://ayuda.pxsol.com/hubfs/image-Jun-25-2020-02-45-22-16-PM.png"
    alt="Listado de servicios Pxsol"
  />
</Frame>

Podrán seleccionar **Ver todos los servicios** si no encuentran algún servicio específico.

<Frame>
  <img
    src="https://ayuda.pxsol.com/hubfs/image-Jun-25-2020-02-47-37-12-PM.png"
    alt="Opción Ver todos los servicios"
  />
</Frame>

Dentro del listado van a encontrar que un mismo servicio puede repetirse redactado de formas diferentes. Por ejemplo: Wifi lo van a encontrar como Wifi libre, Wifi en todo el establecimiento, Wifi en habitaciones, Wifi en zonas comunes, etc.

Es importante que, de cada servicio, elijan solo uno y que sea el que más se adapte al que tienen en el establecimiento, para que no se repita en el listado que muestra la Web.
