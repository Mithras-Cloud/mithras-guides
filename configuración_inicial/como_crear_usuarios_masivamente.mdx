---
title: "Crear usuarios masivamente"
description: "En esta guía explicaremos cómo crear varios usuarios fácilmente utilizando la función de creación múltiple."
---


<Note>
La creación de usuarios solo la puede realizar un usuario que cuente con permiso de administrador. Si su cuenta de usuario no tiene permisos de administrador, necesitará solicitar acceso a un usuario que sí los tenga.
</Note>

<Steps>
<Step title="Acceder a la aplicación Usuarios">
Entra a la aplicación **Usuarios** desde el Omnibox.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-03-2024-01-41-00-4753-AM.png?width=572&amp;height=260&amp;name=image-png-Sep-03-2024-01-41-00-4753-AM.png" alt="Acceso a la aplicación Usuarios" />
</Frame>
</Step>

<Step title="Seleccionar Crear múltiple">
Haz clic en el botón **Crear múltiple**.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-14-2025-03-24-24-3275-PM.png" alt="Botón Crear múltiple" />
</Frame>
</Step>

<Step title="Completar datos del primer usuario">
Se abrirá una ventana para que ingreses los datos del primer usuario (Nombre, Apellido, Email y Contraseña).

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-14-2025-03-25-42-9525-PM.png" alt="Formulario de creación de usuario" />
</Frame>
</Step>

<Step title="Agregar usuarios adicionales">
Selecciona **Agregar usuario** por cada usuario adicional que desees crear y completa sus datos.

<Warning>
El sistema no permitirá generar un usuario con un email que ya se haya utilizado para otro usuario en cualquier empresa de PXSOL.
</Warning>

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-14-2025-03-26-58-2626-PM.png" alt="Botón Agregar usuario" />
</Frame>
</Step>

<Step title="Configurar permisos avanzados (Opcional)">
Si deseas aplicar a todos los usuarios los mismos permisos que un usuario existente, selecciona **OPCIONES AVANZADAS** y luego **Asignar los mismos Permisos de**, eligiendo el usuario correspondiente.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-14-2025-06-22-05-0979-PM.png" alt="Opciones avanzadas de permisos" />
</Frame>

<Info>
Los permisos que seclonan incluyen Menú, Aplicaciones, Aplicaciones de configuración, Cuentas de Correo, Establecimientos, Puntos de venta, Empresas y Equipos Adicionales. No se clonarán permisos de acciones clave como Administrador, Multi-Propiedad, PIN, Visualización de Tarjetas, Restablecimiento de movimientos, Gestión de Informes ni Edición de Tarifas.
</Info>
</Step>

<Step title="Crear usuarios">
Finalmente, selecciona **Crear Usuarios** para finalizar el proceso.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-14-2025-06-26-03-8977-PM.png" alt="Botón Crear Usuarios" />
</Frame>
<Check>
Tus usuarios han sido creados exitosamente.
</Check>
</Step>
</Steps>
