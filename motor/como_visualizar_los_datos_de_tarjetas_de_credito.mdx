---
title: "¿Cómo visualizar los datos de Tarjetas de Crédito?"
description: "En esta guía explicaremos cómo visualizar los datos de tarjetas de crédito de una reserva."
---

<p><strong>1 - </strong>Para visualizar los datos de una tarjeta, es necesario acceder a la reserva correspondiente. Esto se puede hacer a través de:</p>
<ul>
<li>El <strong>Omnibox</strong>, buscando con el ID o nombre del titular correspondiente para ubicar la reserva:&nbsp;</li>
</ul>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-24-2023-03-52-42-6810-PM.png" alt="Omnibox para buscar reservas" />
</Frame>
<ul>
<li>Desde el <strong>Listado de reservas</strong>:</li>
</ul>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-24-2023-03-53-24-0695-PM.png" alt="Listado de reservas" />
</Frame>
<Note>
<p>Para acceder a los datos de las tarjetas, es fundamental que su usuario cuente con el permiso habilitado. Si no dispone de este permiso, le recomendamos que se comunique con un administrador de la propiedad para solicitar su activación. Puede seguir el enlace a continuación para obtener más información sobre cómo habilitar la gestión de tarjetas:</p>
<ul>
<li><strong>&nbsp;<a href="https://ayuda.pxsol.com/es/c%C3%B3mo-habilitar-a-un-usuario" target="_blank" rel="noopener">¿Cómo habilitar a un usuario la gestión de tarjetas de crédito?</a></strong></li>
</ul>
</Note>
<p>&nbsp;</p>
<p><strong>2 - </strong>Una vez que haya localizado la reserva deseada, dirígete a la pestaña <strong>"Tarjeta".</strong></p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-16-03-1808-PM.png" alt="Pestaña Tarjeta en la reserva" />
</Frame>
<p><strong>3 - </strong>Al acceder, se mostrarán las tarjetas vinculadas a la reserva. Para obtener la información completa, simplemente haz clic en el botón "<strong>VER</strong>".</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-18-00-5810-PM.png" alt="Botón VER para mostrar datos de tarjeta" />
</Frame>
<p><strong>4 -</strong> Aparecerá una ventana emergente que les preguntará si desean continuar con la visualización de la tarjeta, proporcionándoles información importante sobre el proceso de visualización.</p>
<Warning>
<p>Los datos de la tarjeta pueden ser consultados un máximo de tres veces y por un lapso de 30 minutos. Una vez que se hayan alcanzado estas tres visualizaciones, la información se eliminará de manera permanente de la base de datos.</p>
</Warning>
<p>Para continuar, presiona el botón <strong>"VISUALIZAR DATOS DE LA TARJETA"</strong>:</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-20-39-1423-PM.png" alt="Botón VISUALIZAR DATOS DE LA TARJETA" />
</Frame>
<p><span style="font-size: 1rem;"><strong>5 -</strong> El sistema presentará los datos de la tarjeta, omitiendo el código de seguridad. Para acceder a este código, simplemente haz clic en la opción <strong>"Mostrar CVV"</strong>.</span></p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-30-47-9673-PM.png" alt="Opción Mostrar CVV" />
</Frame>
<p>Se mostrará el código de seguridad de la siguiente manera:</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-32-56-0295-PM.png" alt="Código de seguridad de la tarjeta" />
</Frame>
<p>Para acceder a los datos de la tarjeta, se implementa un sistema de doble factor de autenticación. Esto significa que, al intentar visualizar una tarjeta, el sistema puede solicitar que ingreses un código que será enviado a tu número de teléfono registrado en los permisos de usuario para la visualización de tarjetas.</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-24-2023-11-49-53-4441-PM.png" alt="Doble factor de autenticación para visualización de tarjeta" />
</Frame>
<Note>
<p>IMPORTANTE!</p>
<p>WhatsApp ha implementado una medida de seguridad que restringe la visualización de los códigos de seguridad al dispositivo principal. Por lo tanto, los códigos no se pueden ver en las versiones de WhatsApp escritorio o web. Para recibir los códigos correctamente, deben tener acceso al teléfono con la aplicación de WhatsApp. Agradecemos su comprensión, ya que estas políticas de seguridad son establecidas por Meta (WhatsApp) y están fuera de nuestro control.</p>
<p>Para obtener más detalles, por favor visita el siguiente enlace: https://faq.whatsapp.com/372839278914311</p>
</Note>
<p>Si intentas acceder a los datos de las tarjetas más veces de las que está permitido, el sistema te notificará con el siguiente mensaje:</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-May-16-2025-06-38-28-3698-PM.png" alt="Mensaje de error por intentos de visualización excedidos" />
</Frame>
