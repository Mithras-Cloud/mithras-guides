---
title: "¿Cómo agregar el código de seguimiento de Google Tag Manager a Pxsol?"
description: "Si ya trabajas con Google Tag Manager, en esta guía aprenderás a vincularlo a Pxsol."
---

<Tip>
  **Google Tag Manager** es un sistema gratuito de gestión de etiquetas que te permite actualizar de forma rápida y sencilla códigos de seguimiento en tu sitio web o motor de reservas. [Más información aquí](https://support.google.com/tagmanager/answer/6102821?hl=es\&ref_topic=3441530).

  Si aún no tienes una cuenta de Google Tag Manager, puedes [crearla aquí](https://marketingplatform.google.com/about/tag-manager/).
</Tip>

<Steps>
  <Step title="Obtener el ID de contenedor de Google Tag Manager">
    Dentro de Google Tag Manager, deberás obtener el ID de tu contenedor.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-40-32-01-PM.png" alt="Obtener ID de contenedor en Google Tag Manager" />
    </Frame>
  </Step>

  <Step title="Acceder a Pxsol">Ingresa a [pms.pxsol.com](https://pms.pxsol.com/).</Step>

  <Step title="Acceder a la aplicación ¨Integraciones¨">
    Haz clic en el buscador (ominibox) y luego selecciona la aplicación "Integraciones".

    <Frame>
      <img height="264" width="530" alt="Imagen" src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/159981281015949874ea536b3d84100d3444e77c8c4.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260430%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260430T172309Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=1a16b7144b3a2b0b5d9a4723f28ac38948f38496753b298ccf93a76967dfce78" />
    </Frame>
  </Step>

  <Step title="En la ventana que se despliega, seleccionar el punto de venta">
    Realiza un scroll hasta encontrar la tarjeta de Google Tag Manager y luego haz clic en el botón de Instalación.

    <Frame>
      <img height="462" width="486" alt="Imagen" src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/35551405327277c7d3b234c1b30ed54e589983939c.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260430%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260430T173106Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=28df35e5a20650f907805ceb191031d4e6cc51143c190c90dba7c2bc1b696f6c" />
    </Frame>
  </Step>

  <Step title="En la ventana que se despliega, seleccionar el punto de venta">
    En el select que muestra la ventana, selecciona el punto de venta en el que tienes instalada la URL a la que debe aplicar el código de GTM.

    <Frame>
      <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/159981376958f9eb3840a142cb84ff4949aed363e79.jpg?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260430%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260430T174055Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=d13abcc9b98e3f26b56beb743bcb8ad5821f6e5e601c99c2b68cbf2407d68c3b" alt="Imagen" />
    </Frame>
  </Step>

  <Step title="Colocar el ID del contenedor de Google Tag Manager">
    Pega el ID de contenedor que obtuviste de Google Tag Manager en el campo correspondiente.

    <Frame>
      <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/35551417401750d190fd8110b0bdda019deef12b12.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260430%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260430T174110Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=a952f4859e11efe74ff220aaa1003cef2421f478217acd0d785f0389da548d47" alt="Imagen" />
    </Frame>
  </Step>

  <Step title="Guardar los cambios">
    Haz clic en el botón para guardar los cambios realizados.

    <Frame>
      <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/28441134944bbb0195cbcad4e99d3dba86c8f7bf916.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260430%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260430T174214Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=c5ce9e20299e306a16c267c22b7c55cd9615b7f43ce28ab9bfdf9ed70ad4c7c6" alt="Imagen" />
    </Frame>
  </Step>
</Steps>

<Warning>
  Si tienes un sitio web y un motor de reservas de Pxsol, este es todo el proceso que debes seguir.
  Si, en cambio, tienes un sitio web de terceros, este proceso solo vinculará tu motor de reservas
  de Pxsol. En ese caso, deberás solicitarle a tu webmaster que vincule GTM a tu sitio web externo.
</Warning>
