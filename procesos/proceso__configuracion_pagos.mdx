---
title: "Proceso - Configuración de Pagos"
description: "En esta guía explicaremos el proceso de configuración para poder registrar pagos."
---

Lo primero que deberás configurar en el hotel serán todas las formas de pago que el hotel maneja. Para eso harás uso de la aplicación Cuentas Contables. En la siguiente guía podrás ver cómo ingresar a la misma y encontrarás información para entender el funcionamiento de esta aplicación:

[¿Cómo visualizar cada Cuenta Contable?](https://ayuda.pxsol.com/como-visualizar-cada-cuenta-contable)

Allí deberás cargar todas las formas de pago en cada moneda que se utilice, siguiendo los pasos de la siguiente guía:

[¿Cómo crear Cuentas Contables?](https://ayuda.pxsol.com/como-crear-cuentas-contables)

Para conocer todas las acciones que se pueden realizar en esa aplicación, recomendamos acceder a nuestro Centro de Ayuda, donde se explicará cómo realizar cada una de estas acciones, ingresando a:

[Guías: Cuentas Contables](https://ayuda.pxsol.com/pms#cuentas-contables)

En esta aplicación podrás llevar un conteo de todos los artículos con los que cuenta el hotel, para aplicar luego a consumos en el folio de una reserva. Permite llevar un control de stock de los productos que se agreguen al folio de las reservas.

Para crear los artículos que utiliza el hotel, sigue los pasos de la siguiente guía:

[¿Cómo cargar inventario?](https://ayuda.pxsol.com/como-cargar-inventario)

Ya configuradas las Cuentas Contables y el Inventario, podrás registrar cada uno de los pagos que reciba el hotel y facturar. Esto permite llevar un control de stock, generar comprobantes y registrar los pagos en una cuenta contable.

Para conocer cómo se realiza el registro de un pago para luego facturar, deberás leer la siguiente guía:

[Proceso - Registrar un Pago y Facturar](https://ayuda.pxsol.com/preceso-registrar-un-pago-y-facturar)

Una vez configuradas las cuentas contables, deberás realizar el mismo procedimiento para las Cuentas Corrientes. Para ingresar a la aplicación y conocer el funcionamiento de la misma, ingresa a la siguiente guía:

[¿Cómo visualizar las Cuentas Corrientes?](https://ayuda.pxsol.com/como-visualizar-las-cuentas-corrientes)

Cuando necesites cargar una nueva Cuenta corriente, lo podrás realizar viendo los pasos de la siguiente guía:

[¿Cómo crear Cuentas Corrientes?](https://ayuda.pxsol.com/como-crear-cuentas-corrientes)

En el Centro de Ayuda encontrarás información para realizar todas las acciones que la aplicación permite.

[Guía: Cuentas Corrientes](https://ayuda.pxsol.com/pms#cuentas-corrientes)

Para llevar un registro de los saldos, pagos y compras que se realizan, deberás agregar cada uno de los proveedores con los que cuenta el hotel. Lo podrás hacer siguiendo los pasos de la siguiente guía:

[¿Cómo cargar Proveedores?](https://ayuda.pxsol.com/como-cargar-proveedores)

Una vez configurados los Proveedores y el Inventario, podrás realizar el proceso del abastecimiento de un Proveedor que involucra la carga de inventario, el registro del pago y la facturación.

Para conocer cómo realizar este proceso, deberás leer la siguiente guía:

[Proceso - Registro de abastecimiento](https://ayuda.pxsol.com/proceso-registro-de-abastecimiento)
