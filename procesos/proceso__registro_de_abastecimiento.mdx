title: "Proceso - Registro de Abastecimiento"
description: "Guía para registrar el abastecimiento de un Proveedor, incluyendo aumento de stock, registro de pago y comprobante."
---


El proceso de abastecimiento involucra los siguientes pasos:

*   Registrar el pago al proveedor.
*   Cargar el comprobante de compra.

### Registrar Pago

Cualquier pago que realices a tu proveedor debe ser registrado en el sistema. Sigue la guía a continuación para registrar el pago y seleccionar la cuenta contable correspondiente desde la cual se emite dicho pago:

<Info>
Para más detalles, consulta la guía: <a href="https://ayuda.pxsol.com/como-registrar-un-pago-de-una-factura-de-proveedor" target="_blank" rel="noopener">¿Cómo registrar un pago de una factura de Proveedor?</a>
</Info>

Una vez que hayas registrado el pago, puedes verificar en la aplicación **Cuentas Contables** que el débito se haya registrado correctamente en la cuenta seleccionada.

<Info>
Para aprender a visualizar las cuentas contables, consulta la guía: <a href="https://ayuda.pxsol.com/como-visualizar-cada-cuenta-contable" target="_blank" rel="noopener">¿Cómo visualizar cada Cuenta Contable?</a>
</Info>

### Cargar Comprobante

Después de registrar el pago de la factura, debes registrar el comprobante proporcionado por tu Proveedor, creando un comprobante de compra. Sigue estos pasos:

<Info>
Para más detalles, consulta la guía: <a href="https://ayuda.pxsol.com/como-cargar-una-factura-desde-proveedores" target="_blank" rel="noopener">¿Cómo cargar una factura desde Proveedores?</a>
</Info>

Al registrar el comprobante, el sistema aumentará automáticamente el stock del artículo ingresado en la cantidad especificada. Puedes corroborar esto accediendo a la aplicación **Inventario**.
