title: "¿Cómo dar permiso o excluir usuarios de un informe?"
description: "En esta guía te contaremos cómo dar permisos a usuarios y equipos para que vean un informe, también como excluir usuarios determinados para que no puedan visualizar un informe."
---


## Permisos de Informes

Dentro de tu lista de informes, al agregar o editar uno, se desplegará una ventana para definir quién tiene permiso para verlo.

<Frame>
<img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-26-36-7793-PM.png" alt="Ventana de configuración de permisos de informe" />
</Frame>

### Opciones de Visibilidad

Puedes configurar la visibilidad del informe de las siguientes maneras:

*   **Todos los usuarios:** Cualquier usuario de la empresa puede visualizar y editar el informe.
<Frame>
<img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-28-21-7076-PM.png" alt="Opción de visibilidad para todos los usuarios" />
</Frame>

*   **Solo yo:** Únicamente tu usuario podrá visualizar y editar el informe.
<Frame>
<img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-29-10-8261-PM.png" alt="Opción de visibilidad solo para mí" />
</Frame>

*   **Personalizar usuarios:** Al seleccionar esta opción, se habilitan campos para personalizar por usuarios o grupos.
    *   Puedes hacer que el informe sea visible para todo un **grupo**.
    <Frame>
    <img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-31-31-7729-PM.png" alt="Selección de grupo para visibilidad del informe" />
    </Frame>
    *   Puedes **seleccionar usuarios específicos**.
    <Frame>
    <img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-33-09-1117-PM.png" alt="Selección de usuarios específicos para visibilidad del informe" />
    </Frame>
    *   Si has seleccionado que todos los usuarios puedan ver el informe (dejando los campos anteriores vacíos) o un grupo, puedes **excluir a determinadas personas** colocándolas en el tercer campo.
    <Frame>
    <img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-35-21-3526-PM.png" alt="Exclusión de usuarios específicos del informe" />
    </Frame>

### Guardar Cambios

Para guardar los cambios realizados, haz clic en el botón **Guardar y Cerrar** en la parte inferior de la ventana.

<img src="https://7398646.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-06-2023-04-36-10-1064-PM.png" alt="Botón Guardar y Cerrar" />
</Frame>
