# Proceso 5 - Visualización de Reporte de Reservas

En esta guía explicaremos el proceso para otorgar permisos de visualización, cómo acceder al reporte y qué encontraremos en él.

### 1. Otorgar Permisos de Visualización a Usuarios

Por defecto, el sistema permite la visualización de todos los informes disponibles según tu plan. Si un usuario no tiene el permiso necesario, deberás configurar el sistema para permitirle el acceso al reporte de Reservas.
