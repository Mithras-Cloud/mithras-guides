# Configuración del Establecimiento para tu Página Web Plan Starter

En esta guía detallaremos los campos del establecimiento que impactarán directamente en la página web de tu plan Starter de pxsol.

## DATOS

### 1. Nombre Público

Es el nombre de tu establecimiento que se mostrará prominentemente en la página de inicio de tu web.

<Frame>
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-30-2021-03-27-07-74-PM.png" alt="Campo Nombre Público en la configuración del establecimiento" />
</Frame>

### 2. Dirección

La dirección que configures aquí aparecerá en el pie de página de tu sitio web.

<Frame>
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-30-2021-03-36-11-99-PM.png" alt="Campo Dirección en la configuración del establecimiento" />
</Frame>

<Note>
Es fundamental que la dirección que ingreses en este campo sea la dirección física de tu hotel para que se muestre correctamente en la página web.
</Note>

### 3. Email de Contacto

Este es el correo electrónico que se mostrará en el pie de página de tu sitio web, permitiendo a los usuarios contactarte fácilmente.

<Frame>
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-30-2021-03-37-55-33-PM.png" alt="Campo Email de Contacto en la configuración del establecimiento" />
</Frame>

<Note>
Al igual que con la dirección, asegúrate de que el email que ingreses sea el correo electrónico oficial de tu hotel para garantizar una comunicación efectiva.
</Note>

## TEXTOS

Desde la pestaña "Textos", tu web tomará únicamente la **descripción larga**. Este contenido será el que se visualice en la página de inicio de tu sitio web.

<Frame>
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-30-2021-03-42-43-22-PM.png" alt="Pestaña Textos en la configuración del establecimiento" />
</Frame>
