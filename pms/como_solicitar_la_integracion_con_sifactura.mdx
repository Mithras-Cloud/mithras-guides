---

<Warning>
  Para llevar a cabo esta integración es necesario utilizar el ERP de SiFactura y estar facturando
  con el facturador de Pxsol.
</Warning>

<Note>
**¿Qué es SiFactura?**

SiFactura es un software de facturación electrónica y contable para Pymes de Argentina.

</Note>

<Note>
**¿Para qué sirve la integración?**

Facilita a los usuarios de SiFactura ver una lista completa de todos los vouchers y recibos generados con nuestro PMS. Mediante este servicio, SiFactura puede acceder a una lista de todos los vouchers asociados con el cliente propietario de la clave API. También permite a SiFactura recuperar y ver todos los detalles de un voucher específico. Al utilizar este servicio, SiFactura puede obtener información detallada de cada voucher, incluyendo datos del cliente y cualquier otra información relevante asociada.

Esto simplifica enormemente la gestión y revisión de todos los vouchers emitidos, proporcionando una visión global y organizada de todas las transacciones. Garantiza que toda la información necesaria para la correcta gestión de los vouchers esté disponible de manera rápida y precisa.

</Note>
