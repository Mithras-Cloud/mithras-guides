---
title: "¿Cómo configurar medios de pago?"
description: "En esta guía explicaremos cómo configurar los diferentes medios de pago que ofrecerán a los huéspedes para realizar la reserva a través del Motor de Reservas."
---


Para ingresar a la aplicación, puedes hacerlo desde el omnibox:

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-17-2024-07-43-05-4319-PM.png" alt="Ingreso a la aplicación desde el omnibox" />
</Frame>

O desde el menú lateral en el apartado **Configuración**, baja hasta **Configuración General** y haz clic en "Medios de pago".

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-04-21-26-58-PM.png" alt="Navegación a Medios de pago en Configuración General" />
</Frame>

Puedes agregar un nuevo medio de pago con el botón **+ Nuevo Medio de Pago**, ubicado arriba a la derecha; o editar uno ya existente desde el **menú de acciones** a la derecha del medio de pago que desees editar.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-17-2024-07-45-32-5620-PM.png" alt="Botón para agregar nuevo medio de pago y menú de acciones" />
</Frame>

<Note>
Para agregar una plataforma de pago como Webpay o Mercado Pago, comunícate con el equipo de Soporte para que te soliciten la información necesaria para gestionar la integración.
</Note>

Se abrirá una ventana lateral con apartados para completar la información necesaria para la configuración:

### 1. Categoría de medio de pago

*   **Categoría**: Despliega el listado de alternativas disponibles.
*   **Nombre único**: Denominación de cómo lo visualizará el huésped en el motor de reservas.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-04-32-47-51-PM.png" alt="Configuración de Categoría y Nombre Único de medio de pago" />
</Frame>

<Note>
Si vinculaste Mercado Pago para que el cliente sea redirigido a dicha plataforma, debes configurarlo como medio de pago la opción **Mercado Pago**. Si deseas cobrar un porcentaje adicional con este medio, completa el porcentaje en el punto **5 - ¿El medio de pago tendrá algún recargo o descuento?**. *NO selecciones la opción Mercado Pago 30%.*
</Note>

### 2. Asociar una plantilla de correo al medio de pago

Puedes seleccionar una plantilla ya creada o crear una nueva plantilla para este medio de pago desde este mismo apartado (ver guía [cómo configurar una plantilla E-Commerce](https://ayuda.pxsol.com/es/como-configurar-las-plantillas-de-correo-e-commerce)).

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-05-19-26-02-PM.png" alt="Asociación de plantilla de correo a medio de pago" />
</Frame>

<Note>
Cada plantilla solo puede estar asociada a una categoría de medio de pago. Por lo tanto, al asignar una plantilla a este medio de pago, se asociará a todos los medios de pago que pertenezcan a esa categoría. Recomendamos crear una plantilla distinta para cada categoría de medio de pago.
</Note>

### 3. Asociar a una política

Este punto mostrará las políticas previamente configuradas para tu establecimiento. Deberás seleccionar una o más políticas para que apliquen a este medio de pago.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-05-28-02-62-PM.png" alt="Asociación de políticas a medio de pago" />
</Frame>

### 4. Definir la moneda para el medio de pago

Podrás seleccionar una o más monedas para este medio de pago; si no seleccionas ninguna, por defecto se aplicará a todas. Seleccionar una moneda sirve para que en el Motor de Reservas solo se muestren las búsquedas que coincidan con la moneda seleccionada en este campo.

Las monedas se configuran desde el **punto de venta**, y en este apartado encontrarás un acceso directo para agregar alguna moneda adicional.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-05-45-08-34-PM.png" alt="Definición de moneda para medio de pago" />
</Frame>

### 5. Definir si este medio de pago tendrá algún tipo de descuento o recargo

*   **Recargo porcentual**: Puedes introducir un porcentaje de incremento para ese medio de pago.
*   **Descuento porcentual**: Puedes introducir un porcentaje de descuento que aplicará al seleccionar este medio de pago.
*   **Cobrar solo un porcentaje**: Si el pago equivale a un porcentaje del total.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-05-50-14-57-PM.png" alt="Definición de recargos o descuentos para medio de pago" />
</Frame>

### 6. Indicar la Ventana de Reserva Mínima y Máxima

Define la diferencia en días entre la fecha de compra y la fecha de ingreso del pasajero al hotel.

<Note>
Por ejemplo, si utilizas transferencia bancaria y no quieres que se realice la transferencia el mismo día del Check-in, puedes colocar **Ventana de reserva mínima: 1 o 2 días**.
</Note>

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-06-25-49-47-PM.png" alt="Configuración de ventana de reserva mínima y máxima" />
</Frame>

### 7. Definir el punto de venta

Si solo tienes un punto de venta, el medio de pago se aplicará a este. En caso de tener más de un punto de venta configurado, puedes seleccionar uno o más puntos de venta para que el medio de pago aplique.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-06-37-22-39-PM.png" alt="Selección de punto de venta para medio de pago" />
</Frame>

### 8. Guardar cambios

Por último, haz clic en el botón **Guardar Cambios** para que el medio de pago quede configurado.

<Frame>
<img src="https://cdn2.hubspot.net/hubfs/7398646/Motor/Medios%20de%20pago/C%C3%B3mo%20configurar%20medios%20de%20pago/7.png" alt="Botón Guardar Cambios" />
</Frame>

En las búsquedas de tarifas en el motor de reservas, podrás visualizar el/los medio(s) de pago creado(s).

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-19-2022-06-40-48-47-PM.png" alt="Visualización de medios de pago en el motor de reservas" />
</Frame>
