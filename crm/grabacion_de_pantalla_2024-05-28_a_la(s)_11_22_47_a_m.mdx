---
title: "Edición de Plantillas de Chat con Contenido Dinámico"
description: "Aprende a personalizar tus plantillas de chat agregando variables dinámicas para una comunicación más efectiva."
---

<Note>
  Esta guía te mostrará cómo editar tus plantillas de chat y personalizarlas con contenido dinámico.
</Note>

<Steps>
  <Step title="Acceder a la sección de edición">
    Para comenzar, localiza la sección de contenido de la plantilla que deseas modificar dentro del formulario "Editar Plantilla". Haz clic sobre el área de contenido, que en este caso se titula "Contenido en español", para abrir la ventana de edición. Esta acción desplegará una interfaz específica donde podrás visualizar y alterar el texto del mensaje.

  </Step>

  <Step title="Posicionar el cursor y abrir el selector de contenido dinámico">
    Dentro de la ventana "Edición contenido plantilla Chat", sitúa el cursor en la posición del texto donde deseas añadir la información dinámica. A continuación, haz clic en el botón "+ Agregar contenido dinámico" para abrir un selector de variables disponibles. Este botón se encuentra sobre el área de texto.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/01/23/screenshot_1769195545_1_6973c819e03dd.jpg" alt="Ventana de edición de contenido de la plantilla de chat con el cursor en el cuerpo del mensaje." />
    </Frame>
  </Step>

  <Step title="Insertar la primera variable dinámica">
    Se abrirá una ventana emergente titulada "Agregar contenido dinámico", que muestra una lista de variables disponibles agrupadas por categorías como "Encabezados", "Reserva" y "Establecimiento". Localiza la categoría "Establecimiento" y haz clic en la variable "Nombre del hotel" para insertarla en tu plantilla. El sistema añadirá automáticamente el código correspondiente a la variable en la posición del cursor y mostrará una notificación de confirmación.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/01/23/screenshot_1769195546_2_6973c81a88ed1.jpg" alt="Selector de contenido dinámico con varias categorías de variables disponibles para insertar." />
    </Frame>
  </Step>

  <Step title="Añadir texto estático y preparar para la siguiente variable">
    Después de insertar una variable, puedes continuar escribiendo texto estático directamente en el editor para complementar la información dinámica. En este caso, se añade una nueva línea y se escribe la frase "Tu link de web CheckIn:" para contextualizar la siguiente variable dinámica que se insertará. Esto permite combinar texto fijo con información que cambiará automáticamente en cada envío.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/01/23/screenshot_1769195547_3_6973c81b14ac1.jpg" alt="Cuerpo del mensaje en el editor de plantillas, mostrando la variable de nombre de hotel y texto estático recién añadido." />
    </Frame>
  </Step>

  <Step title="Insertar la segunda variable dinámica">
    Para añadir otra variable, haz clic nuevamente en el botón "+ Agregar contenido dinámico". En la ventana emergente, busca la categoría "Links" y selecciona la opción "Link Web Check-In". Al hacer clic, la variable correspondiente se insertará inmediatamente después del texto estático que acabas de escribir. El sistema confirmará la inserción con una notificación.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/01/23/screenshot_1769195547_4_6973c81bb25b8.jpg" alt="Ventana de selección de contenido dinámico, con el cursor sobre la variable 'Link Web Check-In' en la categoría 'Links'." />
    </Frame>
  </Step>

  <Step title="Guardar los cambios">
    Una vez que hayas finalizado de añadir todo el contenido dinámico y el texto estático deseado, revisa el mensaje completo en el editor para asegurarte de que tiene el formato correcto. Para finalizar el proceso y aplicar los cambios realizados en la plantilla, haz clic en el botón rojo "GUARDAR Y CERRAR". Esto almacenará la nueva versión de la plantilla y cerrará la ventana de edición.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/01/23/screenshot_1769195548_5_6973c81c623de.jpg" alt="Vista final del contenido de la plantilla con todas las variables insertadas, antes de guardar los cambios." />
    </Frame>
  </Step>
</Steps>
