# Creación de panel personalizado

La aplicación de Informes y Paneles permite crear y personalizar tu propio panel con los gráficos que prefieras para visualizar información de manera rápida y eficiente.

<h4>**¿Cómo crear mi panel?**</h4>
<p>Para configurar un nuevo panel, sigue estos pasos:</p>
<Steps>
<Step title="1. Accede a la aplicación y presiona 'Nuevo panel'">
  Accede a la aplicación Informes y Paneles y presiona "Nuevo panel".

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-23-2025-04-46-16-8321-PM.png" alt="Interfaz de la aplicación para crear un nuevo panel" />
  </Frame>
</Step>

<Step title="2. Configura las opciones generales del panel">
  En el margen derecho, selecciona:
  <ul>
    <li>**Título del panel**</li>
    <li>**Descripción**</li>
    <li>**Paleta de colores** para la visualización de los gráficos</li>
    <li>**Período a visualizar** en la configuración</li>
  </ul>

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-23-2025-04-54-26-5006-PM.png" alt="Opciones de configuración general del panel" />
  </Frame>
</Step>
</Steps>

<p>En el margen izquierdo encontrarás los componentes disponibles para construir tu panel.</p>

<p>**Estructura:**</p>
<ul>
  <li>
    <p>
      Columnas:** Sirven para organizar el panel en distintas secciones. Es necesario agregar al
      menos una columna al arrastrarla hacia el panel, ya que esto habilita la posibilidad de
      insertar gráficos posteriormente.
    </p>
  </li>
  <li>
    <p>
      Caja contenedora:** Permite insertar un recuadro dentro de una columna para agrupar y
      estructurar mejor los elementos.
    </p>
  </li>
</ul>

<p>**Gráficos**:</p>
<p> - Puedes elegir entre gráficos de torta, barras, líneas o combinados (barra y línea).</p>

<p>**Otros**:</p>
<div>** - Datos**: Incluyen KPI, totales, tabla y contadores.</div>

  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-23-2025-04-55-13-2535-PM.png"
    alt="Componentes de datos disponibles para el panel"
  />
</Frame>

<h3 >Construcción del Panel</h3>
<Steps>
  <Step title="1. Estructura el panel con columnas">
    Arrastra y suelta las columnas deseadas para estructurar el panel.
  </Step>
  <Step title="2. Agrega y configura gráficos">
    Agrega y configura los gráficos según tus necesidades.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-23-2025-04-56-11-4374-PM.png" alt="Arrastrar y soltar un gráfico en el panel" />
    </Frame>

  </Step>
  <Step title="3. Selecciona el informe a utilizar">
    Haz clic sobre el recuadro del gráfico y, en el margen derecho, selecciona "Elegir informe a utilizar".

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-03-2025-02-37-22-6893-PM.png" alt="Seleccionar informe para el gráfico" />
    </Frame>

  </Step>
  <Step title="4. Elige el informe">
    Selecciona el informe que deseas visualizar en el gráfico.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-03-2025-02-37-43-4100-PM.png" alt="Lista de informes disponibles para visualizar" />
    </Frame>

  </Step>
  <Step title="5. Personaliza variables">
    Personaliza cada gráfico utilizando diferentes variables.
  </Step>
</Steps>

<h3>Configuración de Gráficos</h3>

<h4>**Gráfico de Torta**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación.</li>
  <li>**Agrupar por**: Criterio para organizar la información.</li>
  <li>**Radio interno**: Personaliza la visualización.</li>
  <li>**Mostrar leyenda**: Habilita la visualización de segmentos.</li>
  <li>**Número máximo de valores**: Limita la cantidad de datos mostrados.</li>
  <li>**Altura máxima**: Define el alto del gráfico.</li>
  <li>**Columna a filtrar**: Permite establecer un filtro en el gráfico.</li>
  <li>
    **Filtrar por**: Campo obligatorio con la variable a visualizar en caso de seleccionar columna a
    filtrar.
  </li>
  <li>
    **Tipo de fecha:** permite determina que variable de fecha se utilizará para mostrar el grafico
    (al igual que sucede en el informe).
  </li>
</ul>

<h4>**Gráfico de Barras**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación.</li>
  <li>**Agrupar por**: Criterio para organizar la información.</li>
  <li>
    **Barras**: Permite configurar las barras que se mostrarán en el gráfico. Incluye:
    <ul>
      <li>
        <em>Columna: </em>Selecciona la columna de datos que se utilizará para generar las barras.
      </li>
      <li>
        <em>Tipo de agregación: </em>Determina cómo se procesarán los datos. Opciones: Suma o
        Conteo.
      </li>
      <li>
        <em>Filtro: </em>Aplica un filtro adicional para limitar los datos.
      </li>
    </ul>
  </li>
  <li>**Mostrar leyenda:** Habilita la visualización de segmentos.</li>
  <li>**Número máximo de valores**: Limita la cantidad de datos mostrados.</li>
  <li>**Altura máxima**: Define el alto del gráfico.</li>
  <li>**Columna a filtrar**: Permite establecer un filtro en el gráfico.</li>
  <li>
    **Filtrar por**: Campo obligatorio con la variable a visualizar en caso de seleccionar columna a
    filtrar.
  </li>
</ul>

<h4>**Gráfico de Líneas**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación.</li>
  <li>**Eje X**.</li>
  <li>
    **Líneas:** Permite configurar las líneas que se mostrarán en el gráfico. Incluye:
    <ul>
      <li>
        <em>Columna: </em>Selecciona la columna de datos que se utilizará para generar las líneas.
      </li>
      <li>
        <em>Tipo de agregación: </em>Determina cómo se procesarán los datos. Opciones: Suma o
        Conteo.
      </li>
    </ul>
  </li>
  <li>
    <em>Filtro: </em>Aplica un filtro adicional para limitar los datos.
  </li>
  <li>**Número máximo de valores**: Limita la cantidad de datos mostrados.</li>
  <li>**Columna a filtrar**: Permite establecer un filtro en el gráfico.</li>
  <li>**Altura máxima**: Define el alto del gráfico.</li>
  <li>**Ordenar**: Ascendente o descendente.</li>
  <li>**Filtrar por**: Campo obligatorio con la variable a visualizar.</li>
</ul>

<h4>**Gráfico Compuesto**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación.</li>
  <li>**Agrupar por**: Criterio para organizar la información.</li>
  <li>
    **Gráficos**: Permite configurar los gráficos que se mostrarán. Incluye:
    <ul>
      <li>Columna: Selecciona la columna de datos que se utilizará para generar los gráficos.</li>
      <li>Tipo de agregación: Opciones: Suma o Conteo.</li>
      <li>Filtro: Aplica un filtro adicional.</li>
    </ul>
  </li>
  <li>**Número máximo de valores**: Limita la cantidad de datos mostrados.</li>
  <li>**Columna a filtrar**: Permite establecer un filtro en el gráfico.</li>
  <li>**Altura máxima**: Define el alto del gráfico.</li>
  <li>**Filtrar por**: Campo obligatorio con la variable a visualizar.</li>
</ul>

<h4>**KPI**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación del gráfico.</li>
</ul>

<h4>**Total**</h4>
<ul>
  <li>**Tipo de agregación**: Método utilizado para calcular el dato.</li>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación del gráfico.</li>
  <li>**Prefijo**: Texto opcional que se mostrará antes del valor.</li>
  <li>**Sufijo**: Texto opcional que se mostrará después del valor.</li>
</ul>

<h4>**Tabla**</h4>
<ul>
  <li>**Número máximo de valores a mostrar:** Limita la cantidad de datos mostrados.</li>
  <li>
    **Columnas a mostrar:** Permite seleccionar los datos del informe que se desean visualizar en la
    tabla.
  </li>
  <li>**Columna de filtro**: Campo utilizado para filtrar los datos.</li>
  <li>**Valor del filtro**: Criterio aplicado para filtrar la información.</li>
</ul>

<h4>**Contador**</h4>
<ul>
  <li>**Título**: Nombre del gráfico.</li>
  <li>**Descripción**: Breve explicación del gráfico.</li>
  <li>**Ícono:** Permite agregar un ícono al título.</li>
  <li>**Color del texto:** Permite seleccionar un color para los datos mostrados.</li>
  <li>
    **Columna:** Permite seleccionar los datos del informe que se desean visualizar en el contador.
  </li>
  <li>
    **Tipo de comparación:** Permite seleccionar un valor fijo de comparación o bien otro dato
    seleccionado columna de comparación.
  </li>
  <li>**Tipo de agregación**: Método utilizado para calcular el dato.</li>
  <li>**Columna de filtro**: Campo utilizado para filtrar los datos.</li>
  <li>**Valor del filtro**: Criterio aplicado para filtrar la información.</li>
</ul>

<h3>Finalización</h3>
<p>Una vez configurados todos los gráficos, haz clic en el botón "Crear panel" para guardarlo.</p>
