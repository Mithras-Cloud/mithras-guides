## Introducción

En esta guía aprenderás a **agregar y configurar el módulo Reviews Pxsol** en tu sitio web construido con el **Editor Web** de Pxsol. Esta funcionalidad te permite mostrar las reseñas de tus huéspedes de manera atractiva y personalizada, mejorando la confianza y la prueba social en tu página.

## Antes de comenzar

Antes de continuar, asegúrate de:

* Tener acceso al módulo **Reviews Pxsol** en tu cuenta de Pxsol.
* Contar con permisos de edición en la página donde deseas agregar el módulo.

## ¿Dónde se encuentra esta función?

Puedes acceder a esta funcionalidad desde:

**Editor Web → Panel de módulos → Reviews Pxsol**

## Cómo utilizar el módulo Reviews Pxsol

Sigue estos pasos:

### 1. Accede al panel de módulos

1. Abre el editor de tu página.
2. Haz clic en el ícono de `(+)` ubicado en la barra de herramientas para abrir el listado de módulos disponibles.

  ![Imagen](https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/212810904363ad9c9be72fa6e6f1f35bf78684e5e51.png)

### 2. Añade el módulo a la página

1. En el panel, escribe **Reviews Pxsol** en el campo de búsqueda.
2. Haz clic en el módulo y arrástralo a la sección de la página donde deseas que se muestren las reseñas.

<Frame>
  <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/19507666651164adf93bf9029c6eb893935d91c1dc2.png" alt="Imagen" />

### 3. Configura el contenido y visibilidad

Una vez añadido el módulo, haz clic sobre él para abrir el panel de configuración. Aquí podrás personalizar el título de la sección, el texto introductorio, y elegir qué detalles de las reseñas se mostrarán.

<Frame>
  <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/35468487186c7f8fc6593061c5a12fa3d6790da033.png" alt="Imagen" />
</Frame>

#### Contenido general:

* **Título de sección:** Define un título como "Lo que dicen nuestros huéspedes".
* **Texto introductorio:** Escribe un breve párrafo invitando a los usuarios a leer las reseñas.

#### Configuración de visibilidad:

Controla qué detalles de cada reseña serán visibles para los visitantes. Puedes activar o desactivar cada elemento con un interruptor:

* **Promedio de puntuaciones:** Muestra una calificación general.
* **Fecha de estadía:** Indica cuándo se alojó el huésped.
* **Nombre del huésped:** Muestra el nombre del huésped.
* **Categorías detalladas:** Desglosa la puntuación por categorías (limpieza, ubicación, etc.).

### 4. Controla los resultados

Filtra las reseñas para mostrar solo las más relevantes:

* **Número máximo de reseñas:** Define cuántas reseñas se mostrarán.
* **Puntuación mínima:** Establece una calificación mínima (ej. 4 de 5 estrellas).

## Resultado esperado

Una vez completado el proceso, deberías ver las reseñas en tu página con la configuración aplicada, mostrando información relevante y filtrada según tus preferencias.

## Consideraciones importantes

* Ten en cuenta que las reseñas se sincronizan con la base de datos de Pxsol, por lo que es posible que las nuevas reseñas aparezcan automáticamente después de un tiempo.
* Si tienes múltiples módulos de reseñas, asegúrate de no duplicar contenido en la misma página.

## Buenas prácticas

* **Actualiza frecuentemente las reseñas** para mantener el contenido fresco.
* **Usa reseñas positivas y específicas** que muestren lo mejor de tu servicio.
