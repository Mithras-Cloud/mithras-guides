---
title: "¿Cómo otorgar acceso a una cuenta de Proveedores a un usuario con permiso básico?"
description: Instrucciones paso a paso para habilitar a un usuario con permiso básico el acceso a una cuenta de Proveedores, permitiéndole visualizarla, cargar facturas y realizar pagos.
---

<Note>
  Todo usuario con permiso de Administrador tiene acceso para visualizar y editar cualquier cuenta
  de proveedores. Solo este tipo de usuario puede habilitar el acceso a las cuentas de proveedores a
  un usuario con permiso básico. Si no se configura este acceso, el usuario básico no podrá
  visualizar ninguna cuenta.
</Note>

<Steps>
<Step title="Habilitar acceso a la aplicación Proveedores para el usuario básico">
1.  El usuario básico debe tener habilitado el acceso a la aplicación **Proveedores**.
2.  Para activarlo, navega a la aplicación **Usuarios**.
3.  Selecciona la opción de ACCIONES del usuario correspondiente y haz clic en EDITAR.
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-28-2025-06-34-31-2890-PM.png" alt="Editar usuario" />
    </Frame>
4.  Ve a "Permisos de acceso al sistema" y selecciona "Aplicaciones".
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-02-2024-01-03-59-5306-AM.png" alt="Permisos de acceso - Aplicaciones" />
    </Frame>
5.  En la sección de Proveedores, verifica que el acceso esté habilitado y configurado como "Todos".
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-02-2024-01-07-38-5866-AM.png" alt="Configurar acceso a Proveedores como 'Todos'" />
    </Frame>
<Check>
El acceso a la aplicación Proveedores está habilitado para el usuario básico.
</Check>
</Step>

<Step title="Acceder a la aplicación Proveedores">
  1. Después de completar la configuración anterior, accede a la aplicación **Proveedores** desde el
  Omnibox o desde el menú principal seleccionando Administración y luego Proveedores.
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-02-2024-01-13-22-5243-AM.png"
      alt="Acceso a Proveedores desde Omnibox"
    />
  </Frame>
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-02-2024-01-14-20-0462-AM.png"
      alt="Acceso a Proveedores desde Menú Principal"
    />
  </Frame>
</Step>

<Step title="Otorgar acceso a una cuenta de Proveedor específica">
  1. Para otorgar acceso a la cuenta del proveedor deseado, selecciona la opción ACCIONES del
  proveedor y luego haz clic en Editar.
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-29-2025-01-43-10-0553-PM.png"
      alt="Editar cuenta de Proveedor"
    />
  </Frame>
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-29-2025-01-43-39-5847-PM.png"
      alt="Opciones de Edición"
    />
  </Frame>
  2. En la sección de Usuarios, elige qué usuario tendrá acceso a esta cuenta de proveedor. Con este
  acceso, el usuario podrá visualizar la cuenta, ver movimientos, cargar facturas y registrar pagos.
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-29-2025-01-44-11-1750-PM.png"
      alt="Seleccionar usuario para acceso a Proveedor"
    />
  </Frame>
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-29-2025-01-44-30-4095-PM.png"
      alt="Lista de usuarios con acceso"
    />
  </Frame>
  3. Selecciona **Guardar Cambios**.
  <Frame>
    <img
      src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-29-2025-01-45-02-3213-PM.png"
      alt="Guardar Cambios"
    />
  </Frame>
</Step>

<Step title="Confirmar los cambios">
1.  Confirma los cambios realizados.
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-02-2024-01-52-28-9173-AM.png" alt="Confirmar cambios" />
    </Frame>
<Check>
El acceso a la cuenta del proveedor ha sido otorgado al usuario seleccionado.
</Check>
</Step>
</Steps>
