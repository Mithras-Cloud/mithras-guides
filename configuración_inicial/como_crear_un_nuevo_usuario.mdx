En esta guía explicaremos cómo crear un nuevo usuario para darle acceso a cada persona que utilice el sistema Pxsol.

## Pasos para crear un nuevo usuario

<Steps>
<Step title="Buscar la aplicación Usuarios">
  Desde el buscador omnibox en la barra superior del sistema, busca la aplicación <strong>Usuarios</strong> y haz clic en ella para ingresar.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-13-2023-12-09-56-1611-PM.png" alt="Buscador omnibox y aplicación Usuarios" />
  </Frame>
</Step>

<Step title="Iniciar la creación de un usuario">
  Una vez dentro del panel de Usuarios, haz clic en el botón <strong>Crear Usuario</strong> ubicado en la parte superior derecha de la aplicación.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-13-2023-12-13-47-0222-PM.png" alt="Botón Crear Usuario" />
  </Frame>
</Step>

<Step title="Ingresar la información del nuevo usuario">
  En el menú que se despliega, completa la siguiente información:
  <ul>
    <li><strong>Usuario:</strong> Introduce una dirección de correo electrónico válida.</li>
    <li><strong>Contraseña:</strong> Debe tener 10 caracteres o más.</li>
    <li><strong>Nombre y Apellido:</strong> Ingresa el nombre completo del usuario.</li>
  </ul>

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-01-2023-05-24-16-7486-PM.png" alt="Formulario de creación de usuario" />
  </Frame>

  <Tip>
  Puedes clonar los permisos de otro usuario existente. Dentro de <strong>Opciones Avanzadas</strong>, utiliza el campo <strong>Asignar los mismos Permisos de:</strong>. Para más detalles, consulta la guía: <a href="https://ayuda.pxsol.com/es/c%C3%B3mo-generar-un-usuario-clonando-los-permisos-de-otro" target="_blank" rel="noopener">¿Cómo generar un usuario clonando los permisos de otro?</a>
  </Tip>
</Step>

<Step title="Confirmar la creación">
  Después de completar la información, haz clic en el botón <strong>Crear usuario</strong>.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-01-2023-05-25-04-8051-PM.png" alt="Botón Confirmar Creación de Usuario" />
  </Frame>

  <Note>
  El sistema no permitirá crear un usuario con una dirección de correo electrónico que ya esté en uso por otro usuario en cualquier empresa dentro de PXSOL.
  </Note>
</Step>

<Step title="Configurar permisos y ajustes adicionales">
  Una vez creado el usuario, puedes acceder al menú de acciones para editar configuraciones y permisos más detallados.

  <Note>
  Algunos permisos, como la visualización de tarjetas de crédito o la habilitación de un PIN de seguridad para acciones restringidas, solo pueden ser configurados por un usuario administrador.
  </Note>
</Step>
</Steps>
