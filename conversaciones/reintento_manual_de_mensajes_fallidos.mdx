Esta guía detalla cómo puedes identificar mensajes que no se enviaron correctamente y cómo reintentar su envío de forma manual.

## Reintentar un Mensaje Fallido

Sigue estos pasos para reenviar un mensaje que no se pudo entregar.

<Steps>
  <Step title="Identifica el mensaje fallido">
    Ingresa a la conversación y localiza el mensaje que muestra el estado **fallido** o un indicador de error.
  </Step>

  <Step title="Inicia el reintento">
    Haz clic en la opción **Reintentar envío** que aparece junto al mensaje. El sistema intentará enviar el mensaje de nuevo.

    <Check>

## Límite de Reintentos

<Info>
  El sistema permite reintentar manualmente el envío de un mensaje hasta **3 veces**. Cada vez que seleccionas "Reintentar envío", el sistema contabiliza el intento de forma automática.
</Info>
