---
title: "Integración con la RM Tool de BENS"
description: "Guía detallada para la integración con la RM Tool de BENS."
---

La herramienta de **Bens** se conecta a nuestro sistema para consultar el **"Informe por noche."**

Actualmente, este informe está disponible en la API de Pxsol a través del endpoint: **company/report/nightly**.

**Flujo de integración**

<Steps>
  <Step title="Solicitud de acceso">
    Bens debe realizar un pedido formal por correo para acceder a la API de Pxsol. Pueden escribir directamente a api@pxsol.com.
  </Step>

  <Step title="Habilitación del endpoint">
    Nuestro equipo de **Tech** activa el acceso correspondiente al endpoint **company/report/nightly**.

    <Check>
      Acceso habilitado para el endpoint company/report/nightly.
    </Check>

  </Step>

  <Step title="Entrega de credenciales">
    Una vez generada la **API key**, se comparte con Bens.

    <Tip>
      Almacenar la API key de forma segura para evitar accesos no autorizados.
    </Tip>

  </Step>

  <Step title="Conexión">
    Con el acceso habilitado y las credenciales proporcionadas, Bens puede avanzar con la integración de su herramienta para consumir el informe por noche.

    <Check>
      La integración se ha completado exitosamente y Bens puede consumir el informe.
    </Check>

  </Step>
</Steps>
