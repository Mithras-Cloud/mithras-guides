# ¿Qué es y para qué sirve Tipos de Habitaciones?

En esta guía explicaremos para qué sirve la aplicación Tipos de Habitaciones.

<p><strong>¿Qué es?</strong></p>
<p>Tipos de habitaciones es una aplicación donde podremos cargar las distintas categorías con las que cuenta el hotel a la hora de ofrecer una habitación. Por ejemplo: Habitaciones Dobles, Habitaciones Triples, etc.</p>
<p>&nbsp;</p>
<p><strong>¿Para qué sirve?</strong></p>
<ul>
<li>Podrá clasificar sus habitaciones según las características con las que cuenta.</li>
<li>Una vez establecidas las categorías podrá asignarle una Tarifa para todas las habitaciones que se vendan con esa clasificación.</li>
<li>Asignarle fotos a todas las habitaciones que tengan las mismas características y coincidan en el Tipo de habitación.</li>
<li>Al presupuestar podrá ver cuántas habitaciones tienen disponibles para cada Tipo de habitación.</li>
<li>Ofrecer en el Motor de Reservas categorías de habitaciones.</li>
</ul>
