---
title: "¿Cómo cambiar el número de WhatsApp para la validación de tarjetas de crédito?"
description: "Aprende a modificar el número de teléfono asociado a la verificación de tarjetas de crédito mediante WhatsApp."
---

<p>
  En esta guía te mostraremos cómo cambiar el número de WhatsApp al que se envía el PIN para validar
  el uso de tus tarjetas de crédito.
</p>

<Steps>
  <Step title="Accede a la aplicación 'Usuarios'">
    Ingresa a la aplicación <strong>Usuarios</strong> a través del Omnibox.

    <Frame>
      <img src="/images/placeholder.png" alt="Captura de pantalla de acceso a la aplicación Usuarios." />
    </Frame>
  </Step>

  <Step title="Selecciona el usuario a editar">
    En el listado de usuarios que aparece, haz clic en el <strong>menú de acciones</strong> (tres
    puntos verticales) a la derecha del usuario que deseas modificar y selecciona la opción{" "}
    <strong>Editar</strong>.

    <Frame>
      <img src="/images/placeholder.png" alt="Captura de pantalla mostrando el menú de acciones y la opción Editar." />
    </Frame>
  </Step>

  <Step title="Deshabilita la Autenticación Multifactor">
    En la ventana que se despliega, desplázate hacia abajo hasta encontrar la sección donde se indica
    que la autenticación de doble factor está activa. Haz clic en el botón{" "}
    <strong>Deshabilitar Autenticación Multifactor para este usuario</strong>.

    <Frame>
      <img src="/images/placeholder.png" alt="Captura de pantalla mostrando el botón para deshabilitar la autenticación multifactor." />
    </Frame>
  </Step>

  <Step title="Vuelve a habilitar la Autenticación Multifactor con el nuevo número">
    Una vez que la autenticación haya sido deshabilitada, procede a habilitarla nuevamente. Durante este proceso, podrás ingresar el nuevo número de WhatsApp que deseas utilizar para recibir los códigos de verificación de tus tarjetas.

    <Tip>
      Consulta la guía para habilitar el doble factor para tarjetas si necesitas un recordatorio del proceso: <a href="https://ayuda.pxsol.com/es/c%C3%B3mo-habilitar-a-un-usuario">Guía para habilitar doble factor para tarjetas</a>.
    </Tip>
  </Step>
</Steps>
