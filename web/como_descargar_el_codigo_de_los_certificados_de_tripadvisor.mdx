---
title: "Cómo descargar el código de los certificados de TripAdvisor"
description: "En esta guía enseñaremos cómo descargar el código para tus certificados y widgets disponibles de TripAdvisor."
---


<Steps>
<Step title="Accede a la sección de Widgets de TripAdvisor">
  Dirígete a la página oficial de <a href="https://www.tripadvisor.com.ar/Widgets" target="_blank" rel="noopener">Widgets de TripAdvisor</a>.
</Step>

<Step title="Busca tu establecimiento">
  En el campo de búsqueda, introduce el nombre de tu hotel y haz clic en el botón <strong>Comenzar</strong>.
</Step>

<Step title="Inicia sesión y visualiza los widgets">
  Ingresa a tu cuenta de TripAdvisor. Una vez dentro, podrás ver todos los widgets disponibles para tu establecimiento.
  <Frame>
    <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Oct-27-2020-03-04-33-85-PM.png" alt="Visualización de widgets disponibles en TripAdvisor para un establecimiento." />
  </Frame>
</Step>

<Step title="Obtén el código del widget">
  Para obtener el código del widget que deseas, haz clic sobre la imagen del mismo. Esto revelará los parámetros configurables (como idioma o cantidad de comentarios) y, debajo, el código HTML correspondiente.
  <Frame>
    <img class="hs-image-align-none" src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Oct-27-2020-03-07-43-04-PM.png" alt="Muestra de parámetros y código HTML de un widget de TripAdvisor." />
  </Frame>
  <Note>
    Puedes copiar y pegar el código personalizado directamente en tu correo electrónico, o hacer clic en <strong>ENVIAR CÓDIGO POR CORREO ELECTRÓNICO</strong> para que se envíe a `equipo@pxsol.com`. Se recomienda usar la opción de envío por correo.</Note>
</Step>
</Steps>
