El web check in permite realizar el tradicional proceso del Check In de manera online. De manera sencilla, tu huésped podrá completar la ficha de registro antes del arribo en la Web App del hotel, ingresando con su número de reserva y apellido o mediante un enlace directo.

### ¿Qué permite el Web Check In?

A través de este proceso, se pueden realizar las siguientes acciones:

- Escanear los documentos del huésped.
- Autocompletar los datos del huésped.
- Firmar digitalmente los términos y condiciones de estadía.

### Ventajas del Check In Online

- **Disminución de contacto:** Reduce el tiempo de contacto entre huéspedes y recepcionistas, cumpliendo con protocolos de seguridad.
- **Eficiencia operativa:** Reduce el tiempo que el personal dedica a la carga manual de datos.
- **Mejor experiencia:** El huésped puede disfrutar de las instalaciones desde el momento de su llegada.
- **Detección temprana:** Permite identificar reservas con riesgo de No Show antes del arribo.

### 1. Envío de comunicación al huésped

El enlace para realizar el Check In online se puede enviar de dos formas:

- **Automática:** El sistema envía un correo automáticamente **48 horas antes** del horario de ingreso, incluyendo detalles de la reserva y el enlace al proceso.
- **Manual:** Se puede enviar por correo o WhatsApp. Para más detalles, consulta la guía: [¿Cómo enviar enlace para Web Check In manualmente?](https://ayuda.pxsol.com/como-enviar-enlace-para-web-check-in-manualmente).

### 2. Acceso a la Web App

El cliente puede acceder a la plataforma de dos maneras:

- **Mediante el enlace enviado:** Lo llevará directamente a la pantalla de su reserva.
- **Mediante Apellido y Número de Reserva:** Útil cuando no se dispone del correo previamente. El huésped ingresa estos datos en la sección de Web Check In del motor de reservas o sitio web.

### 3. Carga de datos por el huésped

Para conocer los pasos específicos que debe seguir el huésped, consulta la guía: [¿Cómo realizar la carga de datos?](https://ayuda.pxsol.com/como-realizar-la-carga-de-datos).

### 4. Notificación al hotel

Una vez completado el proceso, el hotel recibe un aviso por dos vías:

1. **Correo electrónico:** Dirigido a los usuarios configurados para recibir notificaciones.
2. **CRM del sistema:** En la pestaña **Historial** de la reserva aparecerá la confirmación, y en la pestaña **Huéspedes** se podrá ver toda la información cargada.

### 5. Arribo al hotel

Tras recibir la notificación, el hotel solo debe esperar la llegada física del huésped para finalizar el ingreso de ingreso de ingreso de manera ágil.
