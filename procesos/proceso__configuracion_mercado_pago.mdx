---
title: "Proceso: Configuración de Mercado Pago"
description: "Guía para integrar y utilizar la aplicación de Mercado Pago para procesar pagos con tarjetas de crédito y débito."
---


<Info>
Por el momento, la plataforma de pago Mercado Pago solo puede ser habilitada para alojamientos radicados en Argentina.
</Info>

Para poder procesar pagos a través de las tarjetas de crédito o débito de Mercado Pago, deberás seguir los siguientes pasos:

<Steps>
<Step title="1º Paso: Integración con Mercado Pago">
Para realizar la integración, sigue los pasos de la siguiente guía:

[¿Cómo gestionar la integración con Mercado Pago?](https://ayuda.pxsol.com/como-gestionar-la-integracion-con-mercado-pago)

Una vez realizada la integración, podrás observar todos los movimientos dentro de esa aplicación.

<Tip>
Consulta la siguiente guía para ayudarte a ver las transacciones:
[¿Cómo ver las transacciones de Mercado Pago?](https://ayuda.pxsol.com/como-ver-las-transacciones-de-mercado-pago)
</Tip>
</Step>

<Step title="2º Paso: Configuración Medios de Pago">
Asegúrate de que en "Medios de Pago" esté configurado "Pago con tarjeta de crédito (100%)".

Si no tienes esta configuración, deberás crear un nuevo medio de pago, seleccionando la siguiente opción en "Medio de Pago":

<Frame caption="Guía medio de pago Mercado Pago">
<img src="https://f.hubspotusercontent40.net/hubfs/7398646/Guia%20medio%20de%20pago%20Mercado%20pago.png" alt="Guia medio de pago Mercado pago" />
</Frame>

<Tip>
Los pasos para realizar esta configuración se encuentran en la siguiente guía:
[¿Cómo configurar medios de pago?](https://ayuda.pxsol.com/como-configurar-medios-de-pago)
</Tip>
</Step>

<Step title="3º Paso: Cobro con Mercado Pago">
Una vez que tengas la integración y la configuración de medios de pago, estarás listo para realizar cobros con Mercado Pago.

Pueden surgir dos situaciones:
<ul>
<li>Que se cobre con una Tarjeta Nueva.</li>
<li>Que se cobre con una Tarjeta que ya se ha registrado anteriormente.</li>
</ul>

Según el caso, sigue los pasos de la guía correspondiente:
<ul>
<li>[¿Cómo cobrar por Mercado Pago con TC nueva?](https://ayuda.pxsol.com/como-cobrar-por-mercado-pago-con-tc-nueva)</li>
<li>[¿Cómo cobrar por Mercado Pago con TC existente?](https://ayuda.pxsol.com/como-cobrar-por-mercado-pago-con-tc-existente)</li>
</ul>

<Tip>
Para más información sobre beneficios, cargas impositivas, recomendaciones y preguntas frecuentes de Mercado Pago, lee la siguiente guía:
[Información General](https://ayuda.pxsol.com/informaci%C3%B3n-general-mercado-pago)
</Tip>
</Step>
</Steps>
