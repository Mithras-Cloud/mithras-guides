---
title: "¿Cómo activar un widget en la web para recibir consultas dentro de Conversaciones de Pxsol?"
description: En esta guía explicaremos cómo configurar un widget (globo de chat) en la web para que el huésped pueda realizar consultas y que las puedan responder desde la aplicación Conversaciones
---


1. Desde el menú principal, busca dentro de <strong>Conversaciones</strong>, la aplicación <strong>Widget web</strong> y haz clic para abrirla.

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-07-47-47-4935-PM.png" alt="Captura de pantalla: Menú principal de Conversaciones, resaltando la aplicación Widget web." />
   </Frame>

2. Selecciona <strong>Crear Widget</strong>.

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-07-49-17-1828-PM.png" alt="Captura de pantalla: Botón 'Crear Widget' resaltado." />
   </Frame>

3. Se desplegará una ventana con los pasos necesarios para configurarlo. Deberás completar la siguiente información:

   ### PASO 1

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-08-2024-12-39-44-5239-PM.png" alt="Configuración general del widget, primera parte." />
   </Frame>
   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Nov-08-2024-12-40-19-7047-PM.png" alt="Configuración general del widget, segunda parte" />
   </Frame>

   *   <strong>Color:</strong> Selecciona el color de fondo del chat que se desplegará. El color del texto dependerá del color de fondo elegido.
   *   <strong>Ubicación:</strong> Define dónde aparecerá el globo de chat en tu web.
   *   <strong>Nombre:</strong> Es el nombre que se mostrará para identificar a la persona o bot que responderá los mensajes.
   *   <strong>Título de Bienvenida:</strong> Título que aparecerá en el saludo inicial del Widget.
   *   <strong>Mensaje de bienvenida:</strong> Mensaje de bienvenida del Widget que figurará debajo del saludo.
   *   <strong>Datos Adicionales:</strong> Puedes solicitar al huésped el Nombre, Email y Teléfono. Estos datos se usarán para iniciar la comunicación; de lo contrario, la conversación se iniciará con un ID de conversación.
   *   <strong>Selecciona un avatar:</strong> Elige el logo del Hotel o un avatar de las opciones disponibles.

4. Selecciona <strong>Siguiente</strong>.

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-09-14-57-5917-PM.png" alt="Botón 'Siguiente' resaltado para continuar con la configuración." />
   </Frame>

   ### PASO 2

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-10-34-29-3244-PM.png" alt="Configuración para integración del widget." />
   </Frame>

   Existen 2 configuraciones:

   a- El widget utilizado para una web generada por Pxsol:
   Completa los siguientes campos:
   *   Punto de venta al que debe aplicar: El Punto de Venta donde está alojada tu web de Pxsol.
   *   Seleccionar equipo: El equipo al cual se asociarán las conversaciones que ingresen.

   Al guardar los cambios, se mostrará el widget configurado:
   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-10-39-10-4204-PM.png" alt="Vista previa del widget configurado." />
   </Frame>

   b- El widget utilizado para un sitio externo a Pxsol:
   Completa los siguientes campos:
   *   <strong>Dominio</strong>: La URL de la web externa a Pxsol donde se instalará el BOT.
   *   <strong>Seleccionar equipo</strong>: El equipo al cual se asociarán las conversaciones que ingresen.

   Al guardar los cambios, visualizarás el código que deberás enviar al proveedor de la web para que lo implemente y sume el widget.

5. Una vez creado el widget, deberás activarlo para que comience a visualizarse en la web.

   <Frame>
   <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-04-2024-10-42-06-1220-PM.png" alt="Interruptor de activación del widget." />
   </Frame>

6. Los mensajes recibidos a través del widget podrás visualizarlos desde la aplicación de Conversaciones.
