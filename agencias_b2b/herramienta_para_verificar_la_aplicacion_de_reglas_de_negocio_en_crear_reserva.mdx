# Verificar la aplicación de reglas de negocio en Crear Reserva

En esta guía, se explica cómo verificar la aplicación de las reglas de negocio utilizando la funcionalidad disponible en la herramienta Crear Reservas.

## Acceder a la aplicación “Crear Reservas”

Puedes acceder a la aplicación “Crear Reservas” de dos maneras: a través del menú lateral seleccionando el Hub Reservas y luego Crear Reservas, o bien buscando directamente la aplicación Crear Reserva en el omnibox.

## Realizar una simulación de búsqueda

Se abrirá una nueva ventana en la que podrás realizar una simulación de búsqueda dentro de la misma aplicación. Si deseas verificar la aplicación de una regla específica, es fundamental que selecciones el contacto correspondiente, ya que las reglas de negocio pueden variar según el cliente.

<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-24-2025-03-33-04-7164-PM.png" alt="Ventana de simulación de búsqueda en Crear Reservas" />
</Frame>

## Consultar reglas aplicadas a una tarifa

Al consultar las tarifas, encontrarás la opción **“Ver reglas aplicadas a esta tarifa”**. Al hacer clic en este botón, se abrirá una vista detallada que muestra cómo se han aplicado las reglas de negocio a la tarifa que estás visualizando.

<Frame caption="Opción para ver reglas aplicadas a una tarifa">
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-24-2025-03-38-12-6839-PM.png" alt="Botón Ver reglas aplicadas" />
</Frame>

La herramienta proporciona un desglose detallado de las reglas que están influyendo en la búsqueda actual. Estas reglas se alinean con la configuración establecida previamente en la aplicación de Reglas de Negocio.
