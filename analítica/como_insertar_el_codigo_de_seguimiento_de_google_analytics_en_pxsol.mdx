---
title: "¿Cómo insertar el código de seguimiento de Google Analytics en Pxsol?"
description: "En esta guía explicaremos cómo insertar el código de seguimiento de Google Analytics."
---

<Info>
Pxsol puede instalar Google Analytics en tu motor de reservas o diseño web (si son provistos por Pxsol). Sin embargo, la generación de reportes y el análisis de datos serán tu exclusiva responsabilidad.
Si tu sitio web es de terceros, deberás contactar a tu webmaster para vincular Google Analytics.
</Info>

<Note>
Google Analytics solo registrará datos a partir del momento en que el código de seguimiento sea configurado. No recuperará información histórica.
</Note>

### Paso 1: Crear una cuenta de Google Analytics

Tienes dos opciones para crear tu cuenta de Google Analytics:

<Tabs>
<Tab title="Opción A: Que Pxsol la cree por ti">
Envía un correo electrónico a <a href="mailto:equipo@pxsol.com">equipo@pxsol.com</a> con la siguiente información:
*   Nombre deseado para tu propiedad en Google Analytics.
*   La dirección de Gmail con la que deseas que se cree la cuenta.
</Tab>

<Tab title="Opción B: Crear la cuenta tú mismo">
Puedes crear tu cuenta siguiendo la guía oficial de Google:
<a href="https://support.google.com/analytics/answer/1008015?hl=es&amp;ref_topic=3544906">https://support.google.com/analytics/answer/1008015?hl=es&amp;ref_topic=3544906</a>
</Tab>
</Tabs>

### Paso 2: Enviar el ID de seguimiento

Si optaste por crear la cuenta tú mismo (Opción B), deberás enviarnos el ID de seguimiento. Este ID lo obtendrás al finalizar la guía de Google.

Busca una imagen similar a la siguiente para identificar tu ID de seguimiento:

<Frame caption="Ejemplo de ubicación del ID de Seguimiento en Google Analytics">
<img src="https://cdn2.hubspot.net/hubfs/7398646/Google%20Analytics/C%C3%B3mo%20insertar%20el%20c%C3%B3digo%20de%20seguimiento%20de%20Google%20Analytics%20en%20Pxsol/1.png" alt="Ejemplo de ID de Seguimiento de Google Analytics" />
</Frame>

### Paso 3: Acceder a tu analítica web

Una vez que el código de seguimiento esté configurado, podrás acceder a la información de analítica web de tu propiedad iniciando sesión en:

<a href="https://marketingplatform.google.com/about/analytics/">https://marketingplatform.google.com/about/analytics/</a>

Utiliza la misma dirección de Gmail que elegiste o proporcionaste para acceder a tus datos.
