# Cómo acceder a la aplicación Web Check-in

En esta guía explicaremos cómo ingresar a la aplicación Web Check-in y qué podemos encontrar.

## Ingreso a la aplicación

Puedes acceder a la aplicación Web Check-in de dos maneras:

1.  **Desde el menú principal:**
    *   Busca dentro de "Operaciones" la aplicación "Web Check-in".
    *   Haz clic en ella para abrirla.

    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-26-2024-03-20-42-5202-PM.png" alt="Acceso a Web Check-in desde el menú principal" />
    </Frame>

2.  **Mediante el Buscador en Pxsol (Omnibox):**
    *   Escribe "WEB CHECK-IN" en el buscador.
    *   Haz clic en el resultado para abrir la aplicación.

    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-26-2024-03-22-27-2883-PM.png" alt="Acceso a Web Check-in a través del buscador" />
    </Frame>

## Funcionalidades de la aplicación

Al ingresar a la aplicación, encontrarás en la parte superior un resumen de las reservas del día. Estas se diferencian según si el huésped ha completado o no el web check-in.

### Personalizar campos del formulario

Puedes adaptar los campos del formulario de Web Check-in que tus huéspedes deberán completar.

<p>La tabla de configuración de campos incluye 4 columnas:</p>

1.  **Campo:** El nombre del campo a completar.
2.  **Tipo de huésped al que aplica:** Define si el campo debe ser completado por todos los huéspedes, solo por el titular de la reserva, o solo por el acompañante.
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-26-2024-03-51-44-6484-PM.png" alt="Tipos de huésped para campos de formulario" />
    </Frame>
3.  **¿Es obligatorio?:** Indica si el campo es obligatorio. Si un huésped no completa un campo obligatorio, no podrá finalizar el web check-in.
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-26-2024-03-54-21-8861-PM.png" alt="Campo obligatorio en formulario de Web Check-in" />
    </Frame>
4.  **¿Ocultar campo?:** Determina si el campo estará disponible para ser completado por el huésped (Sí/No).
    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-26-2024-03-57-45-2135-PM.png" alt="Ocultar campo en formulario de Web Check-in" />
    </Frame>

Recuerda guardar los cambios al final de la tabla una vez que hayas realizado las modificaciones.
