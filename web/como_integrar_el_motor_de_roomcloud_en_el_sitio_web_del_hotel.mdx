---
title: "¿Cómo integrar el motor de RoomCloud en el sitio web del hotel?"
description: "En esta guía, se explica el proceso para integrar el motor de RoomCloud en el sitio web del hotel, el cual es proporcionado por Pxsol."
---


Para comenzar el proceso, sigue estos pasos:

1 - Envía un correo electrónico a [equipo@pxsol.com](mailto:equipo@pxsol.com) o crea un ticket directamente desde la ticketera, solicitando la vinculación del motor de RoomCloud a la página web proporcionada por Pxsol. Asegúrate de incluir el identificador del hotel en RoomCloud, que corresponde al ID del hotel en la extranet de RoomCloud. Este identificador se encuentra ubicado junto al nombre del hotel:

</Frame>

2 - Espera a recibir la confirmación del equipo de implementación una vez que se haya completado la configuración necesaria.
