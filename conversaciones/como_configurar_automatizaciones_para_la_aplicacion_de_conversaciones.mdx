## Configurar automatizaciones en la aplicación de Conversaciones

Esta guía te mostrará cómo configurar envíos automáticos de plantillas de chat y respuestas automáticas del bot en horarios específicos.

<Steps>
<Step title="Accede a la aplicación de Conversaciones">
  Navega en el menú lateral a **Conversaciones - Conversaciones**. Se abrirá la aplicación en una nueva pestaña.

  <Frame>
  <img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jan-19-2024-07-45-47-5372-PM.png?width=294&amp;height=209&amp;name=image-png-Jan-19-2024-07-45-47-5372-PM.png" alt="Menú lateral de Conversaciones" />
  </Frame>
</Step>

<Step title="Abre los Ajustes">
  Una vez dentro de la aplicación, haz clic en el botón de **Ajustes** ubicado en la esquina inferior izquierda.

  <Frame>
  <img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jul-26-2023-02-46-44-3832-PM.png?width=274&amp;height=181&amp;name=image-png-Jul-26-2023-02-46-44-3832-PM.png" alt="Botón de Ajustes en la esquina inferior izquierda" />
  </Frame>
</Step>

<Step title="Selecciona Automatizaciones">
  Dentro de Ajustes, haz clic en **Automatizaciones**.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-12-26-54-7312-PM.png" alt="Opción Automatizaciones dentro de Ajustes" />
  </Frame>
</Step>

<Step title="Crea una Nueva Automatización">
  Selecciona **Nueva Automatización**.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-12-27-31-2751-PM.png" alt="Botón Nueva Automatización" />
  </Frame>
</Step>

<Step title="Elige el tipo de Automatización">
  Selecciona la acción que deseas automatizar:
  <ul>
    <li><strong>Envío de plantillas:</strong> Permite programar el envío de plantillas de Automatización o Workflow en días y horarios específicos.</li>
    <li><strong>Respuesta automática</strong>: Permite configurar los días y horarios en los que el bot responderá automáticamente.</li>
  </ul>

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-12-28-05-6774-PM.png" alt="Selección del tipo de Automatización" />
  </Frame>
</Step>

<Step title="Configura los detalles de la Automatización">
  Completa el formulario con la información correspondiente. Los campos varían según el tipo de automatización seleccionada:

  <h4>Para envío de Plantillas:</h4>
  <ul>
      <li><strong>Nombre</strong>: Nombre descriptivo para tu automatización.</li>
      <li><strong>Canal</strong>: Selecciona uno o varios canales integrados (Whatsapp, Instagram, Widget de web, etc.).</li>
      <li><strong>Desencadenador:</strong> Define si la automatización se activa tras el primer mensaje o con cada mensaje recibido.</li>
      <li><strong>Plantillas:</strong> Elige las plantillas de tipo Automatización o Workflow. Usa el botón de recarga si no aparecen.</li>
      <li><strong>Workflow:</strong> Configura los rangos de fecha y hora para activar esta automatización.</li>
  </ul>

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-02-01-37-5350-PM.png" alt="Formulario de configuración para envío de plantillas" />
  </Frame>

  <Note>
  Puedes agregar varios rangos de fechas y horarios a una misma automatización.
  </Note>

  <h4>Para respuestas automáticas:</h4>
  <ul>
      <li><strong>Nombre</strong>: Nombre descriptivo para tu automatización.</li>
      <li><strong>Canal</strong>: Selecciona uno o varios canales integrados (Whatsapp, Instagram, Widget de web, etc.).</li>
      <li><strong>Desencadenador:</strong> Define si la automatización se activa tras el primer mensaje o con cada mensaje recibido.</li>
      <li><strong>Bot:</strong> Selecciona el bot configurado en la sección Usuarios que responderá automáticamente.</li>
      <li><strong>Workflow:</strong> Configura los rangos de fecha y hora para activar esta automatización.</li>
  </ul>

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-02-01-37-5350-PM.png" alt="Formulario de configuración para respuestas automáticas" />
  </Frame>

  <Note>
  Puedes agregar varios rangos de fechas y horarios a una misma automatización.
  </Note>
</Step>

<Step title="Guarda la Automatización">
  Haz clic en **Agregar** para guardar tu configuración.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-02-05-11-1516-PM.png" alt="Botón Agregar para guardar la automatización" />
  </Frame>
</Step>

<Step title="Ordena tus Automatizaciones">
  <Warning>
  El orden de las automatizaciones determina la secuencia de ejecución.
  </Warning>
  Puedes reordenar las automatizaciones arrastrando los puntos a la izquierda del nombre de cada una.

  <Frame>
  <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-14-2024-02-06-58-1397-PM.png" alt="Reordenamiento de automatizaciones" />
  </Frame>
</Step>
</Steps>
