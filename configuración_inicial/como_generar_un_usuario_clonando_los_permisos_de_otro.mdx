---
title: "¿Cómo generar un usuario clonando los permisos de otro?"
description: "En esta guía explicaremos cómo crear un nuevo usuario utilizando la configuración de permisos de un usuario ya creado."
---


<Note>
La creación de usuarios sólo la puede realizar un usuario con permiso de administrador. Si tu cuenta de usuario no tiene permisos de administrador, necesitarás solicitar acceso a un usuario que sí los tenga.
</Note>

1. Entra a la aplicación **Usuarios** desde el Omnibox.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Sep-03-2024-01-41-00-4753-AM.png?width=572&height=260&name=image-png-Sep-03-2024-01-41-00-4753-AM.png" alt="Interfaz de la aplicación Usuarios" />
</Frame>

2. En el listado de usuarios, haz clic en el menú de acciones a la derecha del usuario cuyos permisos deseas clonar y selecciona la opción "Clonar Usuario".

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-30-2025-12-24-05-3398-PM.png" alt="Menú de acciones y opción Clonar Usuario" />
</Frame>

3. En la ventana que aparece, ingresa la información solicitada:
    *   **Nombre y Apellido:** del nuevo usuario.
    *   **Mail:** un correo electrónico que no se haya utilizado previamente.
    *   **Contraseña:** de 10 caracteres o más, que cumpla con los requisitos de seguridad.

4. Selecciona **Crear Usuario**.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jun-30-2025-12-27-42-3967-PM.png" alt="Botón Crear Usuario" />
</Frame>

<Warning>
El usuario generado tendrá la misma configuración de permisos que el usuario del cual se clonó. Se clona la configuración de los siguientes permisos:
<ul>
<li>Menú</li>
<li>Aplicaciones</li>
<li>Aplicaciones de configuración</li>
<li>Cuentas de Correo</li>
<li>Establecimientos</li>
<li>Puntos de venta</li>
<li>Empresas</li>
<li>Equipos Adicionales</li>
</ul>
Los siguientes permisos NO se clonarán:
<ul>
<li>Administrador</li>
<li>Multi-Propiedad</li>
<li>PIN</li>
<li>Visualización de Tarjetas</li>
<li>Restablecimiento de movimientos</li>
<li>Gestión de Informes</li>
<li>Edición de Tarifas</li>
</ul>
</Warning>
