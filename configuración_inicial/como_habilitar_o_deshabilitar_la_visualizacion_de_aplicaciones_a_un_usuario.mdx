---
title: "¿Cómo habilitar o deshabilitar la visualización de aplicaciones a un usuario?"
description: "En esta guía explicaremos cómo permitir o no que un usuario pueda visualizar ciertas aplicaciones."
---


<Note>
La habilitación de aplicaciones para un usuario sólo puede ser realizada por un usuario con permisos de administrador.
</Note>

<p>&nbsp;</p>

<Steps>
<Step title="Accede a la aplicación 'Usuarios'">
  Ingresa a la aplicación <strong>Usuarios</strong> desde el Omnibox.

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-05-2022-03-32-52-63-PM.png" alt="Captura de pantalla del Omnibox con la aplicación Usuarios seleccionada."/>
  </Frame>
</Step>

<Step title="Selecciona la opción 'Editar' para el usuario">
  En el listado de usuarios, haz clic en el menú de acciones (tres puntos verticales) a la derecha del usuario que deseas editar y selecciona la opción <strong>Editar</strong>.

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-05-2022-03-34-38-70-PM.png" alt="Captura de pantalla del menú de acciones de un usuario con la opción 'Editar' seleccionada."/>
  </Frame>
</Step>

<Step title="Configura el acceso a las aplicaciones">
  En la ventana que se despliega, navega hasta la sección de configuración y haz clic en el apartado <strong>Aplicaciones</strong>.

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jan-12-2024-05-10-43-4242-PM.png" alt="Ventana de configuración de usuario mostrando la sección de Aplicaciones."/>
  </Frame>

  Visualizarás un listado de aplicaciones donde podrás configurar los permisos de cada usuario (Ver, Editar o Borrar). Las opciones disponibles son:
  <ul>
    <li><strong>Todos:</strong> Permite Ver, Editar o Borrar todo lo disponible en la aplicación.</li>
    <li><strong>Mi equipo:</strong> Permite Ver, Editar o Borrar sólo lo asociado al equipo del usuario dentro de la aplicación (no todas las aplicaciones soportan esta opción).</li>
    <li><strong>Propios:</strong> Permite Ver, Editar o Borrar sólo lo asociado a este usuario dentro de la aplicación (no todas las aplicaciones soportan esta opción).</li>
    <li><strong>Nadie:</strong> No permite Ver, Editar o Borrar nada dentro de la aplicación.</li>
  </ul>

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jan-12-2024-05-12-48-0956-PM.png" alt="Listado de aplicaciones con opciones de permisos para el usuario."/>
  </Frame>

  <Note>
  No todas las aplicaciones están desarrolladas para admitir esta configuración completa. Por el momento, podrás configurar la visualización completa o no de las aplicaciones, utilizando la configuración de Ver (Todos o Nadie) para cada aplicación.
  </Note>
</Step>

<Step title="Guarda los cambios">
  Una vez que hayas configurado las aplicaciones que el usuario debe ver o no ver, haz clic en el botón <strong>Guardar Cambios</strong>.

  <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-05-2022-03-47-10-17-PM.png" alt="Botón de 'Guardar Cambios' en la ventana de edición de usuario."/>
  </Frame>

  <Check>
  Los permisos de visualización de aplicaciones para el usuario se han actualizado correctamente.
  </Check>
</Step>
</Steps>
