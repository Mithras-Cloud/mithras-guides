---
title: "¿Cómo cargar Base de Conocimiento desde una conversación?"
description: "En esta guía explicaremos cómo adicionar una pregunta y respuesta para darle información al BOT de Conversaciones, desde una conversación con un Huésped."
---

**1-** Accede desde el **menú lateral** a la aplicación **Conversaciones**

<Frame>
  <img
    src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jul-26-2023-04-12-33-4255-PM.png?width=371&height=208&name=image-png-Jul-26-2023-04-12-33-4255-PM.png"
    alt="Acceso a la aplicación Conversaciones"
  />
</Frame>

**2-** Una vez dentro de la aplicación, haz clic en el menú lateral en el apartado **Clientes**

<Frame>
  <img
    src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jul-26-2023-04-20-33-2975-PM.png?width=247&height=229&name=image-png-Jul-26-2023-04-20-33-2975-PM.png"
    alt="Apartado Clientes"
  />
</Frame>

<Note>
**Importante**, para utilizar la parte de chat con clientes es necesario tener vinculado un número de WhatsApp. Por el momento, la vinculación la realiza el equipo de adquisición. Aparecerá un cartel con el enlace para comunicarse con ventas.

<Frame>
<img src="https://ayuda.pxsol.com/hubfs/image-png-Jul-26-2023-08-42-12-2744-PM.png" alt="Vinculación de número de WhatsApp" />
</Frame>
</Note>

**3-** Las nuevas conversaciones entrantes llegarán a la carpeta **Sin asignar**

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-55-47-7233-PM.png"
    alt="Carpeta Sin asignar"
  />
</Frame>

**4-** Dentro de la conversación, selecciona el **desplegable** de la pregunta o respuesta que deseas cargar en la Base de Conocimiento. Luego, selecciona **Base de Conocimiento**.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-59-44-0831-PM.png"
    alt="Seleccionar Base de Conocimiento"
  />
</Frame>

**5-** Se abrirá una ventana lateral donde se cargará la Pregunta y Respuesta seleccionada, corregida por IA. Selecciona la Base de Conocimiento donde se cargará esta información. Corrige las Preguntas y Respuestas a tu gusto.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-59-22-5779-PM.png"
    alt="Ventana lateral para cargar Pregunta y Respuesta"
  />
</Frame>

**6-** Selecciona **Agregar**.

<Frame>
  <img
    src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-07-01-49-5874-PM.png"
    alt="Botón Agregar"
  />
</Frame>

<Note>
  La Pregunta y Respuesta se cargará como un nodo de información en la aplicación Base de
  Conocimiento y servirá para futuras preguntas similares.
</Note>
