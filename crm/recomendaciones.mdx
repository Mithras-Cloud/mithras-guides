---
title: "Recomendaciones para el Registro de Consultas"
description: "En esta guía encontrará recomendaciones para la aplicación registro de consultas"
---

**¿Qué busca?**

Que uses el sistema para registrar todas las consultas que recibe el hotel.

**¿Por qué es importante?**

Al registrar las consultas telefónicas en el sistema, podrás no sólo aumentar la base de datos sino también centralizar las respuestas y seguimiento para mejorar tus ventas.

De esta manera le mostrarás interés a tu potencial huésped y lograrás su preferencia para concretar reservas y evitar que el huésped reserve por otro canal de venta, o incluso otro hotel!

**Sugerencias**

- Registra todas las llamadas y consultas que recibas en el Registro de consultas para poder responderlas desde el CRM Hotelero.
- Asegúrate de que todo tu equipo conozca el procedimiento para registrar consultas.
- Si no puedes responder en el momento las consultas, utiliza las funciones de "Registrar" o "Agendar".
- Una vez respondidas las consultas, mueve las mismas a la instancia de "Lista de espera" para gestionarlas con recordatorios y campañas de email marketing automation.
- Asegúrate de activar los recordatorios automáticos para fomentar que el potencial huésped responda tus correos y de esa manera puedas avanzar en el proceso de venta.
- Agrega las consultas que recibas por WhatsApp para responderlas a través del sistema.
