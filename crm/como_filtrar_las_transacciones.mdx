---
title: "¿Cómo filtrar las transacciones?"
description: "En esta guía explicaremos cómo buscar transacciones en las instancias, en base a distintos filtros."
---

<Steps>
  <Step title="Acceder a las instancias">
    Ingresa a cualquiera de las instancias para aplicar el filtro y ordenar las transacciones.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Jan-31-2021-11-58-58-81-PM.png" alt="Captura de pantalla mostrando cómo acceder a una instancia." />
    </Frame>

    El sistema te mostrará el listado de todas las reservas en la instancia seleccionada.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Feb-01-2021-12-02-50-37-AM.png" alt="Listado de reservas en una instancia." />
    </Frame>

  </Step>

  <Step title="Aplicar filtros">
    Para filtrar las transacciones, define la información en la barra superior. Puedes especificar algunos o todos los campos:

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Feb-01-2021-12-00-52-90-AM.png" alt="Barra de filtros para transacciones." />
    </Frame>

    <Info>
      Detalle de los campos disponibles para filtrar:

      - **Carpetas:** Define si buscar en todas las instancias o en alguna en particular (consulta, preventa, venta, check in, etc.).
      - **Tipo de fecha:** Selecciona el tipo de fecha para la búsqueda (check in, check out, consulta o última acción).
      - **Desde:** Especifica el rango de fecha de inicio para la búsqueda.
      - **Hasta:** Especifica el rango de fecha de fin para la búsqueda.
      - **Registros:** Determina la cantidad de transacciones a mostrar por página.
      - **Orden:** Define el criterio de ordenación (automático, fecha límite de bloqueo, fecha última acción, fecha ingreso, fecha egreso, fechas cancelación o habitación).
      - **Ocupación:** Filtra por transacciones con habitación asignada, sin asignar, o solo sin asignar.
    </Info>

    Al seleccionar los campos, el sistema filtrará y mostrará los resultados automáticamente.

  </Step>

  <Step title="Eliminar filtros">
    Si deseas eliminar el filtro aplicado, presiona el botón <kbd>QUITAR FILTROS</kbd>.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Feb-01-2021-12-07-54-25-AM.png" alt="Botón para quitar filtros." />
    </Frame>

  </Step>

  <Step title="Ver informe de transacciones">
    Para visualizar el filtro aplicado en formato informe (para descargar o imprimir), sigue la guía detallada en el siguiente enlace:

    </Info>

    <Check>
      Has aprendido a buscar y filtrar transacciones dentro de tus instancias.
    </Check>

  </Step>
