---
title: "¿Cómo solicitar la integración con SiFactura?"
description: "En esta guía explicaremos los pasos a realizar para poder solicitar la integración con el ERP SiFactura."
---


<Warning>
Para llevar a cabo esta integración es necesario utilizar el ERP de SiFactura y estar Facturando con el Facturador de Pxsol.
</Warning>

<Note>
<p><strong>¿Qué es SiFactura?</strong></p>
<p>Si Factura es un Software de Facturación Electrónica y Contable para Pymes de Argentina.</p>
</Note>

<Note>
<p><strong>¿Para qué sirve la integración?</strong></p>
<p>Facilita a los usuarios de SI Factura ver una lista completa de todos los vouchers y recibos generados con nuestro PMS. Mediante este servicio, SI Factura puede acceder a una lista de todos los vouchers asociados con el cliente propietario de la clave API. También, permite a SI Factura recuperar y ver todos los detalles de un voucher específico. Al utilizar este servicio, SI Factura puede obtener información detallada de cada voucher, incluyendo datos del cliente y cualquier otra información relevante asociada.&nbsp;<br><br>Esto simplifica enormemente la gestión y revisión de todos los vouchers emitidos, proporcionando una visión global y organizada de todas las transacciones. Garantiza que toda la información necesaria para la correcta gestión de los vouchers esté disponible de manera rápida y precisa.</p>
</Note>
