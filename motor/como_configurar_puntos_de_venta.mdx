Aquí te guiaremos a través del proceso de configuración de un Punto de Venta y la información necesaria para cargar.

## Acceder a la configuración de Puntos de Venta

1.  Abre el buscador del Panel de Inicio haciendo clic en él o presionando `ALT + S`.

    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-23-2024-05-52-53-8557-PM.png" alt="Buscador del Panel de Inicio" />
    </Frame>

2.  Escribe "Puntos de venta" en el buscador y selecciona la opción correspondiente.

    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-23-2024-05-53-08-5750-PM.png" alt="Buscador de Puntos de Venta" />
    </Frame>

3.  Una vez dentro, presiona "Acciones" sobre el Punto de Venta que deseas modificar y selecciona "Editar información".

    <Frame>
    <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Sep-23-2024-05-53-48-3517-PM.png" alt="Editar información del Punto de Venta" />
    </Frame>

## Completar la información del Punto de Venta

### Configuración

*   **Nombre del punto de venta**: Ingresa el nombre de tu establecimiento.
*   **Idioma por defecto**: Define el idioma principal en el que se mostrará la web y el motor de reservas.
*   **Idiomas disponibles**: Selecciona los idiomas que el huésped podrá elegir.

### Datos del Establecimiento

*   Teléfono / Teléfono 2
*   Whatsapp
*   Página de Facebook (Ej: `https://www.facebook.com/Usuario`)
*   Perfil de Instagram (Ej: `https://www.instagram.com/Usuario`)
*   Perfil de X (Twitter) (Ej: `https://www.x.com/Usuario`)
*   Perfil de TikTok (Ej: `https://www.tiktok.com/Usuario`)
*   Canal de YouTube (Ej: `https://www.youtube.com/Usuario`)
*   Página de LinkedIn (Ej: `https://www.linkedin.com/Usuario`)
*   Link de TripAdvisor (Ej: `https://www.tripadvisor.com/Usuario`)
*   **Moneda por defecto**: Establece la moneda principal que se mostrará en el motor de reservas.
*   **Monedas disponibles**: Define las monedas que el huésped podrá seleccionar.
*   **Tarjetas de crédito recibidas**: Indica las tarjetas de crédito que acepta tu establecimiento.

<Note>
El teléfono principal es obligatorio. Debe incluir el código de país y el código de área (ejemplo para Argentina - Mendoza: `+54 261`).
</Note>

### Personalización y Canal de Venta

*   **Color principal**: Ingresa el código hexadecimal del color principal. Puedes usar [este conversor](https://convertacolor.com/) para transformar colores CMYK o RGB a HEX.
*   **Logotipo**: Sube tu logotipo en formato JPG o PNG. Se recomienda usar PNG con fondo transparente.
*   **Isotipo (logo / símbolo)**: Carga el icono que aparecerá en la pestaña del navegador (favicon). Se recomienda usar PNG con fondo transparente, formato cuadrado y un tamaño de 250px x 250px.

### Buscador

*   **Cantidad máxima de habitaciones**: Define el número máximo de habitaciones que los usuarios pueden seleccionar en una reserva.
*   **Venta de reservas del selector de fechas**: Establece cuántos días a partir de la fecha actual se bloqueará el calendario en el motor de reservas.

### Plantillas de Correo

(Edición de la cabecera y pie de cada email enviado)

*   Cabecera
*   Pie de página

### Analítica y Seguimiento

*   **Google Analytics**: Ingresa el código de seguimiento para analizar el comportamiento del huésped en tu sitio web.
*   **Google Tag Manager**: Ingresa el código de seguimiento para monitorear acciones específicas del huésped en tu sitio web.
*   **Google Conversion Label (búsquedas)**: Introduce el ID de la conversión para búsquedas.
*   **Google Conversion Label (reservas)**: Introduce el ID de la conversión para reservas.
*   **Facebook Pixel ID**: Tu ID de Pixel de Facebook.
*   **Facebook Page ID**: Tu ID de página de Facebook.

## Guardar Cambios

5.  Presiona "Guardar cambios" para aplicar la configuración.
