---
title: "¿Cómo realizar gráficos en Informes?"
description: "En esta guía explicaremos cómo generar gráficos con la información que brinda un informe."
---


Para generar gráficos con la información de un informe, sigue estos pasos:

<Steps>
<Step title="1. Abrir el buscador">
  Abre el buscador del Panel de Inicio haciendo clic en él o presionando `ALT + S`.

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Apr-01-2021-12-43-35-99-PM.png" alt="Buscador del Panel de Inicio" />
  </Frame>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Apr-01-2021-12-42-44-95-PM.png" alt="Ventana del buscador" />
  </Frame>
</Step>

<Step title="2. Filtrar informes">
  En la columna de **Reportes**, visualiza los informes disponibles. Escribe en el buscador "informe de reservas" para filtrar y seleccionar el informe deseado.

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image_png-Apr-07-2021-05-41-13-81-PM.png" alt="Filtrando informe de reservas" />
  </Frame>
</Step>

<Step title="3. Abrir el informe">
  Haz clic en el informe para abrir una nueva pestaña en tu navegador con el formato del informe.

  <Frame>
  <img src="https://ayuda.pxsol.com/hubfs/image-585.png" alt="Formato del informe" />
  </Frame>
</Step>

<Step title="4. Seleccionar columnas relevantes">
  Para generar gráficos de manera efectiva, destilda todas las columnas que no aporten información relevante para tu gráfico. Por ejemplo, si deseas un gráfico que muestre las tarifas totales por instancia en el informe de Reservas, mantén visibles solo las columnas "Instancia" y "Total Neto".

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-610.png" alt="Selección de columnas" />
  </Frame>
</Step>

<Step title="5. Agrupar datos">
  Realiza las agrupaciones de datos pertinentes. Siguiendo el ejemplo anterior, agrupa por "Instancia" y muestra la sumatoria del "Total Neto" para cada instancia.

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-611.png" alt="Agrupación de datos" />
  </Frame>
</Step>

<Step title="6. Ordenar la información">
  Ordena la información para ajustar la presentación del gráfico. Por ejemplo, ordena "Total Neto" de menor a mayor para que el gráfico se muestre de forma ascendente.
</Step>

<Step title="7. Seleccionar datos para graficar">
  Una vez que tengas la información lista para graficar, selecciona todas las celdas que la contienen.

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-612.png" alt="Selección de celdas para graficar" />
  </Frame>
</Step>

<Step title="8. Generar el gráfico">
  Haz clic derecho dentro del área seleccionada y navega a "Graficar este rango". Se desplegarán las opciones de gráficos disponibles. Selecciona el tipo de gráfico que deseas crear.

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-613.png" alt="Opciones de gráficos" width="469" />
  </Frame>

  Tienes disponibles los siguientes tipos de gráficos:
  <ul>
  <li><strong>Gráficos de columna</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-614.png" alt="Gráfico de columna" width="310" />
  </Frame>
  </li>
  <li><strong>Gráficos de barras</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-615.png" alt="Gráfico de barras" width="313" />
  </Frame>
  </li>
  <li><strong>Gráficos de torta</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-617.png" alt="Gráfico de torta" width="320" />
  </Frame>
  </li>
  <li><strong>Gráficos de Línea</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-618.png" alt="Gráfico de línea" width="330" />
  </Frame>
  </li>
  <li><strong>Gráficos de dispersión</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-621.png" alt="Gráfico de dispersión" width="324" />
  </Frame>
  </li>
  <li><strong>Gráficos de Zona</strong>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-619.png" alt="Gráfico de zona" width="322" />
  </Frame>
  </li>
  </ul>
</Step>

<Step title="9. Configurar o descargar el gráfico">
  Una vez generado el gráfico, podrás configurarlo o descargarlo. Coloca el cursor sobre el gráfico para ver las opciones que aparecen a la derecha ("Configuración" y "Descargar").

  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-623.png" alt="Opciones del gráfico" />
  </Frame>

  <ul>
  <li><strong>Configuración:</strong> Permite personalizar la visualización del gráfico, modificar datos, añadir títulos, ejes, leyendas, etc.
  <Frame>
  <img src="https://ayuda.pxsol.com/hs-fs/hubfs/Screenshot%202020-06-10%20at%2011-20-45.png?width=62&amp;name=Screenshot%202020-06-10%20at%2011-20-45.png" alt="Icono de configuración" width="62" />
  </Frame>
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-624.png" alt="Panel de configuración del gráfico" />
  </Frame>
  </li>
  <li><strong>Descargar:</strong> Al hacer clic, se descargará una imagen del gráfico en formato PNG.
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-625.png" alt="Icono de descarga" width="47" />
  </Frame>
  </li>
  </ul>

  <Note>
  Si los datos seleccionados no son adecuados para el tipo de gráfico elegido, este se presentará de la siguiente manera:
  <Frame>
  <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-622.png" alt="Gráfico con datos no válidos" width="584" />
  </Frame>
  </Note>
</Step>
</Steps>
