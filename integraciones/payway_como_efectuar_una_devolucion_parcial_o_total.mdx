---
title: "Payway: Cómo efectuar una devolución parcial o total"
description: "Guía paso a paso para realizar devoluciones de pagos procesados con Payway desde una reserva en Pxsol."
---

Esta guía te explica cómo realizar una devolución (parcial o total) de un pago procesado a través de Payway.

<Note>El monto mínimo para efectuar una devolución con Payway es de **$50 pesos argentinos**.</Note>

## Requisitos previos

- Tener acceso a la reserva con el pago acreditado
- Contar con un PIN de usuario configurado en el sistema

## Pasos para realizar la devolución

<Steps>
<Step title="Acceder a la reserva">
Dirígete a la reserva donde se encuentra acreditado el pago que deseas devolver.
</Step>

<Step title="Ir a la solapa de pagos">
  Dentro de la reserva, ubícate en la sección o solapa de **Pagos**.
</Step>

<Step title="Seleccionar la opción de devolución">
Sobre el pago ya acreditado, haz clic en **Acciones** y elige **Efectuar devolución**.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/undefined-Dec-03-2025-08-59-00-0894-PM.png" alt="Menú de acciones del pago mostrando la opción Efectuar devolución" />
</Frame>
</Step>

<Step title="Completar los datos de la devolución">
Se abrirá una ventana donde deberás completar:

- **Motivo de la devolución**: Describe la razón de la devolución
- **Tipo de devolución**:
  - **Total**: El sistema muestra automáticamente el importe completo del pago
  - **Parcial**: Se habilita un campo para indicar el monto específico a devolver

<Warning>
Recuerda que el monto mínimo para devoluciones con Payway es de $50 ARS.
</Warning>
</Step>

<Step title="Ingresar el PIN del usuario">
Para confirmar la devolución, ingresa tu PIN de usuario. Cualquier usuario con PIN configurado puede realizar devoluciones.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/undefined-Dec-03-2025-08-59-00-7079-PM.png" alt="Campo para ingresar el PIN de usuario y confirmar la devolución" />
</Frame>
</Step>

<Step title="Confirmar la operación">
Una vez confirmada, el sistema genera un nuevo registro en movimientos de pago con:

- El importe devuelto
- El detalle del usuario que realizó la operación

  />

<Check>
La devolución se ha procesado correctamente y queda registrada en el historial de movimientos de pago.
</Check>
</Step>
</Steps>

<Tip>
  Puedes consultar el historial de movimientos de pago en cualquier momento para verificar el estado
  de las devoluciones realizadas.
</Tip>
