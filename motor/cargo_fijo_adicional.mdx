---
title: "Creación de un Nuevo Cargo Obligatorio (Fee) para el motor de reservas"
description: "Aprenda a configurar y agregar cargos adicionales obligatorios a las reservas, como tasas de limpieza o costos administrativos, utilizando un proceso guiado con simulador en tiempo real para verificar el impacto."
---

Esta funcionalidad le permite configurar y agregar cargos adicionales obligatorios a las reservas, como tasas de limpieza, costos administrativos u otros servicios fijos. El proceso guiado le permitirá definir el nombre, la forma de cálculo, el importe y las condiciones específicas bajo las cuales se aplicará el cargo. Se incluye un simulador en tiempo real para verificar cómo el cargo afectará el total de la reserva según la configuración seleccionada.

<Steps>
  <Step title="Iniciar la creación de un nuevo cargo">
    Para iniciar la creación de un nuevo cargo, haga clic en el botón **+ Nuevo cargo**, ubicado en la esquina superior derecha de la sección **Impuestos y extras**. Esta acción abrirá un asistente de configuración en una ventana emergente.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829343_0_6a43d11f5fde7.jpg" alt="Captura de pantalla de la interfaz de 'Impuestos y extras' con el botón '+ Nuevo cargo' resaltado." />
    </Frame>
  </Step>

  <Step title="Seleccionar el tipo de cargo">
    En el primer paso del asistente, **¿Qué tipo de cargo es?**, seleccione la naturaleza del cargo. Elija la opción **Fee obligatorio**.

    <Note>
      Esta opción se utiliza para cargos como limpieza o costos administrativos que se aplican de manera fija según las reglas establecidas.
    </Note>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829344_1_6a43d1203d69a.jpg" alt="Captura de pantalla de la ventana emergente 'Nuevo cargo' con 'Fee obligatorio' seleccionado." />
    </Frame>
  </Step>

  <Step title="Avanzar al siguiente paso">
    Una vez seleccionado el tipo de cargo, presione el botón **Continuar** para avanzar al siguiente paso de la configuración.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829344_2_6a43d1208e4f6.jpg" alt="Captura de pantalla del botón 'Continuar' resaltado en la ventana emergente." />
    </Frame>
  </Step>

  <Step title="Nombrar el cargo">
    En el segundo paso, **¿Cómo se llama el cargo?**, ingrese un nombre descriptivo para el nuevo cargo en el campo de texto **¿Cómo se mostrará este cargo?**. Este nombre será visible para el huésped en el detalle de la reserva.

    Por ejemplo, ingrese **Fee de limpieza**.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829345_3_6a43d1213393a.jpg" alt="Captura de pantalla del campo de texto para el nombre del cargo con 'Fee de limpieza' ingresado." />
    </Frame>
  </Step>

  <Step title="Agregar una descripción (Opcional)">
    Opcionalmente, puede agregar una descripción más detallada sobre el cargo en el campo **¿Quieres agregar una descripción?**. Este texto se mostrará como información adicional en el carrito de reservas.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829345_4_6a43d1219bacf.jpg" alt="Captura de pantalla del campo de texto para la descripción del cargo con 'Prueba' ingresada." />
    </Frame>
  </Step>

  <Step title="Continuar a la configuración de aplicación">
    Tras completar el nombre y la descripción opcional, presione nuevamente el botón **Continuar** para proceder a la configuración de la aplicación del cargo.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829346_5_6a43d1222db26.jpg" alt="Captura de pantalla del botón 'Continuar' presionado." />
    </Frame>
  </Step>

  <Step title="Definir el modelo de aplicación del cargo">
    El tercer paso, **¿Cómo se aplica el cargo?**, es donde define el modelo de cálculo. Se presentan varias opciones para determinar cómo se cobrará el cargo. Las opciones principales son **Por noche**, **Por estadía** y **Por habitación**.

    Adicionalmente, puede especificar si el cobro se realiza por persona, seleccionando entre **Por huésped**, **Por adulto** o **Por niño**.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829346_6_6a43d122b4d8d.jpg" alt="Captura de pantalla de la sección de configuración de aplicación del cargo, mostrando opciones como 'Por noche', 'Por estadía' y 'Por habitación'." />
    </Frame>
  </Step>

  <Step title="Establecer el importe del cargo y usar el simulador">
    En la misma pantalla, defina el importe del cargo en el campo **¿Cuál será el cargo aplicado?**. Puede establecer un monto fijo o un porcentaje.&#x20;

    <Note>
      A la derecha, el simulador de motor de reservas actualiza el total de la reserva en tiempo real para reflejar cómo este nuevo cargo impacta el precio final. Utilícelo para verificar su configuración.
    </Note>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829347_7_6a43d1230efc9.jpg" alt="Captura de pantalla del campo de importe con '5' ingresado y el simulador de reservas actualizado." />
    </Frame>
  </Step>

  <Step title="Ajustar el cálculo y verificar con el simulador">
    Para ajustar el cálculo, modifique la configuración. En lugar de **Por estadía**, seleccione **Por habitación**. El simulador de la derecha se actualizará instantáneamente para mostrar el nuevo cálculo.

    <Tip>
      Esto le permite visualizar el impacto de cada cambio en la configuración antes de guardarla, asegurando que el cargo se aplique como espera.
    </Tip>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829347_8_6a43d123a48e1.jpg" alt="Captura de pantalla del usuario cambiando la selección de 'Por estadía' a 'Por habitación' y el simulador actualizándose." />
    </Frame>
  </Step>

  <Step title="Especificar la inclusión de impuestos">
    Más abajo en la misma pantalla, especifique si el importe del cargo ya incluye los impuestos o si estos deben sumarse por separado. Se presentan dos opciones: **Sí, incluye impuestos** y **No, los impuestos se suman**.

    <Info>
      Al seleccionar la segunda opción, el sistema calculará y sumará los impuestos configurados sobre la base de este cargo.
    </Info>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829348_9_6a43d1245761a.jpg" alt="Captura de pantalla de las opciones para incluir impuestos, con 'Sí, incluye impuestos' seleccionado." />
    </Frame>
  </Step>

  <Step title="Avanzar a las condiciones de aplicación">
    Una vez definido el modelo de cálculo, haga clic en **Continuar** para pasar a la siguiente etapa, donde puede establecer condiciones de aplicación.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829348_10_6a43d124c8f93.jpg" alt="Captura de pantalla del botón 'Continuar' para avanzar al paso de condiciones." />
    </Frame>
  </Step>

  <Step title="Agregar condiciones de aplicación (Opcional)">
    En el cuarto paso, **¿Alguna condición?**, puede definir reglas específicas para limitar cuándo se aplicará este cargo. Para ello, haga clic en el botón **+ Agregar condición**.

    <Note>
      Si no agrega ninguna condición, el cargo se aplicará siempre que esté activo.
    </Note>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829349_11_6a43d1253f097.jpg" alt="Captura de pantalla de la pantalla para agregar condiciones, con el botón '+ Agregar condición' resaltado." />
    </Frame>
  </Step>

  <Step title="Elegir tipo de condición: por ejemplo Nacionalidad del huésped">
    Al agregar una condición, se despliega una lista de opciones disponibles.&#x20;

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829349_12_6a43d12585f2b.jpg" alt="Captura de pantalla del menú emergente de tipos de condiciones, con 'Nacionalidad del huésped' seleccionado." />
    </Frame>
  </Step>

  <Step title="Avanzar a la revisión final">
    Con las condiciones ya definidas, presione el botón **Continuar** para avanzar al último paso del asistente.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829350_14_6a43d126955a8.jpg" alt="Captura de pantalla del botón 'Continuar' para ir a la pantalla de revisión final." />
    </Frame>
  </Step>

  <Step title="Revisar la configuración del cargo">
    El paso final, **Revisión**, muestra un resumen completo de toda la configuración del cargo. Verifique el tipo de cargo, el nombre, el modelo de cálculo, el importe y las condiciones aplicadas.

    <Note>
      Cada sección cuenta con un botón **Editar** que le permite volver al paso correspondiente para realizar ajustes si es necesario.
    </Note>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829350_15_6a43d126e228e.jpg" alt="Captura de pantalla de la pantalla de revisión con un resumen de toda la configuración del nuevo cargo." />
    </Frame>
  </Step>

  <Step title="Guardar y salir">
    Si toda la información en la pantalla de revisión es correcta, finalice el proceso haciendo clic en el botón **Guardar y salir**.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829351_16_6a43d127b87d8.jpg" alt="Captura de pantalla del botón 'Guardar y salir' presionado." />
    </Frame>
  </Step>

  <Step title="Verificar la creación del cargo">
    Al guardar, la ventana emergente se cierra y el sistema regresa a la pantalla principal de **Impuestos y extras**. Un mensaje de confirmación **Cargo creado correctamente** aparecerá brevemente.

    <Check>
      El nuevo **Fee de limpieza** ahora figura en la lista de cargos, dentro de la sección correspondiente según su configuración de impuestos.
    </Check>

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/30/screenshot_1782829352_17_6a43d12839914.jpg" alt="Captura de pantalla de la tabla de 'Fees' mostrando el 'Fee de limpieza' recién creado." />
    </Frame>
  </Step>
</Steps>
