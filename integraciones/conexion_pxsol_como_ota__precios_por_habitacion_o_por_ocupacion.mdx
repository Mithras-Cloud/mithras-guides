<Info>
  Para propiedades que trabajan con precios por ocupación, así como también con precios por
  habitación, esta mejora permite conectar Pxsol como OTA con Roomcloud.
</Info>

Dentro de Roomcloud, el conector a utilizar es **Pxsol XML RC**. El usuario a asociar a la cuenta de Roomcloud es **gui-pxsol**. La tarjeta que se debe habilitar en canales de venta es: **PMS Externo via roomcloud**.
