La funcionalidad de **Gestión de Claves de API** le permite instalar el módulo de integraciones generales y generar claves de API (API Keys). Estas claves son **tokens de acceso seguros** que permiten a sistemas externos conectarse e interactuar con la plataforma del hotel de forma autorizada, heredando los permisos del usuario asociado a la clave.

Usted puede crear múltiples claves, cada una con un conjunto de permisos específicos para segmentar el acceso según la necesidad de la integración.

## Instalación del Módulo de Integraciones

Para habilitar la generación y gestión de claves de API, primero debe instalar el módulo correspondiente.

<Steps>
  <Step title="Diríjase al Marketplace de Integraciones">
    Vaya a la sección de **"Integraciones"** o **"Marketplace"** de su sistema.

    </Frame>
  </Step>

  <Step title="Localice e instale el módulo de API Keys">
    Busque la opción **"Integraciones Generales (API Keys)"** dentro de las integraciones disponibles y seleccione **"Instalar"**.

    <Frame caption="Cursor del mouse sobre el botón 'Instalar' de la tarjeta 'Integraciones Generales (API Keys)'.">

    Marque la casilla de aceptación y presione **"Instalar"** para continuar.

    <Warning>
      Lea atentamente los términos y condiciones, ya que la instalación de este módulo podría implicar costos asociados al servicio.
    </Warning>

## Creación de una Nueva Clave de API

Desde la pantalla de gestión **"Mis API Keys"**, usted puede generar nuevas claves para diferentes integraciones.

<Steps>
  <Step title="Inicie la creación de una nueva clave">
    Haga clic en el botón **"Crear API Key"** para iniciar el proceso de creación de una nueva clave.

    <Frame caption="Cursor del mouse haciendo clic en el botón rojo 'Crear API Key'.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/29/screenshot_1782767867_4_6a42e0fbd729a.jpg" alt="Captura de pantalla del botón 'Crear API Key' resaltado, indicando la acción de hacer clic." />
    </Frame>
  </Step>

  <Step title="Configure los detalles de la clave">
    Complete el formulario **"Crear nueva API Key"** con los detalles de su nueva clave.

    <Frame caption="Formulario 'Crear nueva API Key' con campos como Nombre y Usuario asociado.">
      ![Captura de pantalla del formulario 'Crear nueva API Key', mostrando los campos para introducir la información de la clave.](https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/06/29/screenshot_1782767868_5_6a42e0fc427b7.jpg)
    </Frame>

    Debe definir los siguientes campos:

    * **Nombre:** Asigne un nombre descriptivo que le permita identificar el propósito de la clave, por ejemplo, "API key de prueba para CRM".
    * **Usuario asociado:** Seleccione un usuario existente del sistema. La clave de API operará con los permisos de este usuario. Todas las acciones realizadas a través de la API se registrarán como si las hubiera hecho el usuario seleccionado.
    * **Expiración:** Establezca un período de validez para la clave.
      <Tip>
        Se recomienda seleccionar un período de expiración definido (por ejemplo, "12 meses") para mejorar la seguridad y requerir una rotación periódica de claves.
      </Tip>
    * **Alcance:** Defina a qué recursos tendrá acceso la clave.
      * **Implícito:** Otorga acceso a todos los recursos de la propiedad actual.
      * **Explícito:** Le permite restringir el acceso a empresas y establecimientos específicos, lo cual es útil en entornos con múltiples propiedades.
    * **Entorno:** Seleccione **"Producción"** para que la clave sea funcional en el ambiente operativo real.

    **Defina los permisos específicos:**
    Despliegue cada sección (por ejemplo, "Hotel", "Reservas") y active los interruptores para conceder las autorizaciones necesarias. Esto le permite un control granular sobre las acciones que la integración externa podrá realizar, como "Consultar disponibilidad", "Ver datos del hotel", "Crear reservas" o "Listar reservas".
