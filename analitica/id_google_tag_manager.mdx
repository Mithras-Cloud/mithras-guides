## Configuración del ID de Google Tag Manager

Esta guía te guiará a través del proceso de instalación y configuración de la integración con Google Tag Manager (GTM). El objetivo es vincular un punto de venta específico con su correspondiente ID de contenedor de GTM, permitiendo el seguimiento de eventos y la gestión eficiente de etiquetas de marketing y análisis.

<Steps>
  <Step title="Acceder a la sección de Integraciones">
    Utiliza la barra de búsqueda en la esquina superior izquierda para buscar "Integraciones".

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/05/05/screenshot_1778009517_0_69fa45adc9de5.jpg" alt="Barra de búsqueda en la esquina superior izquierda." />
    </Frame>

    Escribe "int" en la barra de búsqueda y selecciona "Integraciones" de los resultados.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/05/05/screenshot_1778009517_1_69fa45adee0f4.jpg" alt="Resultados filtrados en la barra de búsqueda." />
    </Frame>

    <Check>
      Serás redirigido a la página de integraciones.
    </Check>
  </Step>

  <Step title="Instalar la integración de Google Tag Manager">
    Busca la tarjeta de Google Tag Manager en la sección de integraciones.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/05/05/screenshot_1778009518_3_69fa45ae823ed.jpg" alt="Vista del catálogo de integraciones, con la integración de Google Tag Manager." />
    </Frame>

    Haz clic en el botón rojo "Instalar".

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/05/05/screenshot_1778009518_4_69fa45aeba722.jpg" alt="Botón Instalar de la integración de Google Tag Manager." />
    </Frame>

    <Check>
      Aparecerá una ventana emergente para confirmar la instalación.
    </Check>
  </Step>

  <Step title="Configurar la integración">
    En la ventana emergente, selecciona tu punto de venta del menú desplegable.

    <Frame>
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2026/05/05/screenshot_1778009519_6_69fa45af6327f.jpg" alt="Panel de configuración de Google Tag Manager con un menú desplegable para seleccionar el punto de venta." />
    </Frame>

    Introduce el ID de tu contenedor de Google Tag Manager en el campo de texto.

    Haz clic en el botón "GUARDAR".

    <Frame>
      <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/1066806152474b5fff460b236e304c398ff7bf8fc9f.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260505%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260505T194414Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=c78b62ab67934680faab8edacddc603ca9099b5449063fd012a646f09a982ec9" alt="Imagen" />
    </Frame>

    <Check>
      La integración con Google Tag Manager está configurada.
    </Check>

    <Warning>
      Si cuentas con el sitio web y el motor de reservas de Pxsol, este es el proceso completo que debes seguir.
    </Warning>
  </Step>
</Steps>
