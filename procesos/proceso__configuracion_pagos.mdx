---
title: "Proceso - Configuración de Pagos"
description: "En esta guía explicaremos el proceso de configuración para poder registrar pagos."
---


<p>Lo primero que deberemos configurar en el hotel serán todas las formas de pago que el hotel maneja. Para eso haremos uso de la aplicación Cuentas Contables. En la siguiente guía podrán ver cómo ingresar a la misma y encontrarán información para entender el funcionamiento de esta aplicación:</p>
<p><a href="https://ayuda.pxsol.com/como-visualizar-cada-cuenta-contable" rel="noopener noreferrer">¿Cómo visualizar cada Cuenta Contable?</a></p>

<p>Allí deberán cargar todas formas de pago en cada moneda que se utilice, siguiendo los pasos de la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/como-crear-cuentas-contables" rel="noopener noreferrer">¿Cómo crear Cuentas Contables?</a></p>

<p>Para conocer todas las acciones que se pueden realizar en esa aplicación recomendamos acceder a nuestro Centro de Ayuda, donde se explicará cómo realizar cada una de estas acciones, ingresando a:</p>
<p><a href="https://ayuda.pxsol.com/pms#cuentas-contables" rel="noopener noreferrer">Guías: Cuentas Contables</a></p>

<p>En esta aplicación podrá llevar un conteo de todos los artículos con los que cuenta el hotel, para aplicar luego a consumos en el folio de una reserva. Permite llevar un control de stock de los productos que se agreguen al folio de las reservas.</p>
<p>Para crear los artículos que utiliza el hotel seguir los pasos de la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/como-cargar-inventario" rel="noopener noreferrer">¿Cómo cargar inventario?</a></p>

<p>Ya configuradas las Cuentas Contables y el Inventario, podrán registrar cada uno de los pagos que reciba el hotel y facturar. Esto permite llevar un control de stock, generar comprobantes y registrar los pagos en una cuenta contable.</p>

<p>Para conocer cómo se realiza el registro de un pago para luego facturar deberá leer la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/preceso-registrar-un-pago-y-facturar" rel="noopener noreferrer">Proceso - Registrar un Pago y Facturar</a></p>

<p>Una vez configuradas las cuentas contables deberá realizar el mismo procedimiento para las Cuentas Corrientes. Para ingresar a la aplicación y conocer el funcionamiento de la misma ingresar a la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/como-visualizar-las-cuentas-corrientes" rel="noopener noreferrer">¿Cómo visualizar las Cuentas Corrientes?</a></p>

<p>Cuando necesite cargar una nueva Cuenta corriente lo podrá realizar viendo los pasos de la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/como-crear-cuentas-corrientes" rel="noopener noreferrer">¿Cómo crear Cuentas Corrientes?</a></p>

<p>En el Centro de Ayuda encontrará información para realizar todas las acciones que la aplicación permite.</p>
<p><a href="https://ayuda.pxsol.com/pms#cuentas-corrientes" rel="noopener noreferrer">Guía: Cuentas Corrientes</a></p>

<p>Para llevar un registro de los saldos, pagos y compras que se realizan, deberá agregar cada uno de los proveedores con los que cuenta el hotel. Lo podrá hacer siguiendo los pasos de la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/como-cargar-proveedores" rel="noopener noreferrer">¿Cómo cargar Proveedores?</a></p>

<p>Una vez configurados los Proveedores y el Inventario, podremos realizar el proceso del abastecimiento de un Proveedor que involucra la carga de inventario, el registro del pago y la facturación.</p>
<p>Para conocer cómo realizar este proceso deberá leer la siguiente guía:</p>
<p><a href="https://ayuda.pxsol.com/proceso-registro-de-abastecimiento" rel="noopener noreferrer">Proceso - Registro de abastecimiento</a></p>
