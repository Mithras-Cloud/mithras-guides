# Payway: Preguntas Frecuentes

En esta guía veremos las preguntas frecuentes que pueden surgir con respecto a la integración con Payway.

## 1 - ¿Cómo obtener las credenciales de Payway solicitadas por Pxsol?

Para obtener las credenciales, es necesario contar con un establecimiento habilitado para <strong>e-commerce o links de pago</strong> y con acceso al <strong>portal de ventas online de Payway</strong>.

Cumplidos estos requisitos, podés solicitar las credenciales directamente al equipo de soporte de Payway. Ellos te enviarán un correo, como se muestra a continuación, con el asunto <strong>“Datos de Producción”</strong>, donde encontrarás la información necesaria:

<ul>
<li aria-level="1"><strong>Site ID</strong></li>
<li aria-level="1"><strong>API Key Pública</strong></li>
<li aria-level="1"><strong>API Key Privada</strong></li>
</ul>

Estas credenciales son las que Pxsol requiere para configurar correctamente la integración.

Si no llega el correo, puedes ingresar al "Portal de Payway Ventas Online (Decidir Plus)" y dirigirte a la sección de “Configuración” - “Mis ApiKeys” en el menú principal.
