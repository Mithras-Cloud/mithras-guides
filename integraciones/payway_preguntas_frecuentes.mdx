# Payway: Preguntas Frecuentes

En esta guía veremos las preguntas frecuentes que pueden surgir con respecto a la integración con Payway.

## 1 - ¿Cómo obtener las credenciales de Payway solicitadas por Pxsol?

Para obtener las credenciales, es necesario contar con un establecimiento habilitado para **e-commerce o links de pago** y con acceso al **portal de ventas online de Payway**.

Cumplidos estos requisitos, podés solicitar las credenciales directamente al equipo de soporte de Payway. Ellos te enviarán un correo, como se muestra a continuación, con el asunto **"Datos de Producción"**, donde encontrarás la información necesaria:

- **Site ID**
- **API Key Pública**
- **API Key Privada**

Estas credenciales son las que Pxsol requiere para configurar correctamente la integración.
