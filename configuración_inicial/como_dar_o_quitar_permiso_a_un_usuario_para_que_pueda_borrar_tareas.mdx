---
title: "¿Cómo dar o quitar permiso a un usuario para que pueda borrar Tareas?"
description: "En esta guía te enseñaremos cómo habilitar a un usuario para que pueda borrar tareas de la lista en la aplicación TAREAS del sistema."
---

<Note>
  La habilitación de aplicaciones para un usuario **solo la puede realizar un usuario que cuenta con
  permiso de administrador.** Si tu usuario no es administrador, deberás solicitar el acceso que
  cuente con ese permiso.
</Note>

<Steps>
  <Step title="Ingresar a la aplicación Usuarios">
    Entra a la aplicación **Usuarios** desde el Omnibox.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-21-2023-01-46-48-6774-PM.png" alt="Acceso a la aplicación Usuarios" />
    </Frame>

  </Step>

  <Step title="Editar usuario">
    En el listado de usuarios que aparece, haz clic en el menú de acciones a la derecha del usuario que deseas editar y selecciona la opción **Editar**.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-21-2023-01-47-29-0760-PM.png" alt="Menú de acciones y opción Editar" />
    </Frame>

  </Step>

  <Step title="Localizar permisos">
    En la ventana que se despliega, desplázate hacia abajo hasta la sección **Permisos**. Busca el ítem que dice **Habilitar permiso para borrar Tareas**.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-31-2023-01-29-24-0114-PM.png" alt="Sección Permisos" />
    </Frame>

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-31-2023-01-30-45-1131-PM.png" alt="Opción Habilitar permiso para borrar Tareas" />
    </Frame>

  </Step>

  <Step title="Configurar permiso">
    Utiliza el interruptor (switch) de la derecha para habilitar o deshabilitar este permiso para el usuario que estás editando.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-31-2023-01-31-15-8205-PM.png" alt="Interruptor de permiso" />
    </Frame>

  </Step>

  <Step title="Guardar cambios">
    Una vez que hayas configurado los permisos correctamente, haz clic en el botón **Guardar y Cerrar**.

    <Frame>
      <img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Oct-31-2023-01-31-57-0634-PM.png" alt="Botón Guardar y Cerrar" />
    </Frame>

  </Step>
</Steps>
