---
title: "Cambiar o agregar tarjeta de crédito para pagar Pxsol via Stripe"
description: "Instrucciones para actualizar o añadir una tarjeta de crédito para pagos a través de Stripe."
---

<Steps>
  <Step title="Acceder al enlace de administración">
    Haz clic en el enlace enviado por administración. Este enlace será similar a la siguiente imagen:

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Feb-19-2021-06-44-54-99-PM.png" alt="Ejemplo de enlace de pago" />
    </Frame>

  </Step>

  <Step title="Completar datos de la tarjeta">
    Completa los datos de tu tarjeta de crédito y haz clic en **pagar**:
    
    *   Número de tarjeta
    *   Fecha de vencimiento
    *   Código de seguridad (CVV)

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Feb-19-2021-06-47-01-36-PM.png" alt="Formulario de pago con tarjeta de crédito" />
    </Frame>

  </Step>
</Steps>
