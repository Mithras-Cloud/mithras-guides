# ¿Cómo facturar desde Plano de Mesas?

En esta guía explicaremos como generar un comprobante para facturar los consumos de una mesa del Plano de Mesas.

### 1. Acceder al Plano de Mesas

Ingresa mediante el uso del "Buscador en Pxsol" (Omnibox) y escribe **PLANO DE MESAS**.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Feb-19-2024-07-53-48-2765-PM.png?width=572&amp;height=125&amp;name=image-png-Feb-19-2024-07-53-48-2765-PM.png" alt="Buscador Pxsol con 'PLANO DE MESAS'" />
</Frame>

Luego, ingresa a **Plano de Mesas**.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Feb-19-2024-07-54-20-1989-PM.png?width=419&amp;height=218&amp;name=image-png-Feb-19-2024-07-54-20-1989-PM.png" alt="Opción 'Plano de Mesas' en el buscador" />
</Frame>

Alternativamente, puedes acceder desde **Servicios - Plano de Mesas**.
<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Mar-14-2024-03-00-21-2959-PM.png?width=266&amp;height=404&amp;name=image-png-Mar-14-2024-03-00-21-2959-PM.png" alt="Menú de servicios con 'Plano de Mesas'" />
</Frame>

### 2. Visualizar las Mesas

La aplicación mostrará distintos cuadros. Cada cuadro corresponde a una mesa o barra del hotel.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Mar-14-2024-03-01-04-2336-PM.png?width=672&amp;height=499&amp;name=image-png-Mar-14-2024-03-01-04-2336-PM.png" alt="Plano de Mesas con cuadros representando mesas" />
</Frame>

### 3. Editar la Mesa

Edita la mesa para la cual deseas generar el comprobante.

<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Apr-23-2024-03-03-36-3575-PM.png?width=432&amp;height=261&amp;name=image-png-Apr-23-2024-03-03-36-3575-PM.png" alt="Seleccionar una mesa para editar" />
</Frame>

### 4. Generar Comprobante

Selecciona **Generar Comprobante** y luego **Confirmar**.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-11-2024-01-55-18-0746-PM.png" alt="Opciones de mesa con 'Generar Comprobante'" />
</Frame>

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-11-2024-01-55-48-4512-PM.png" alt="Confirmar la generación del comprobante" />
</Frame>

### 5. Completar Datos de Facturación

Se desplegará una ventana lateral donde podrás cargar los datos necesarios para la facturación.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-11-2024-01-57-22-2248-PM.png" alt="Formulario de datos de facturación" />
</Frame>

<Tip>
Para conocer los pasos detallados sobre cómo completar este formulario y emitir el comprobante, te recomendamos revisar la siguiente guía: <a href="https://ayuda.pxsol.com/es/c%C3%B3mo-facturar-desde-el-m%C3%B3dulo-facturaci%C3%B3n-nuevo" target="_blank" rel="noopener">¿Cómo facturar desde el módulo Facturación?</a>
</Tip>

### 6. Ver Factura

Una vez facturado, podrás ver el PDF generado haciendo clic en **Ver Factura**.

<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Jul-11-2024-02-01-07-2316-PM.png" alt="Opción 'Ver Factura' después de generar el comprobante" />
</Frame>
