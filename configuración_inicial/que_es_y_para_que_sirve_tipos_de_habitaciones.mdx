# ¿Qué es y para qué sirve Tipos de Habitaciones?

En esta guía explicaremos para qué sirve la aplicación Tipos de Habitaciones.

**¿Qué es?**

Tipos de habitaciones es una aplicación donde podrás cargar las distintas categorías con las que cuenta el hotel a la hora de ofrecer una habitación. Por ejemplo: Habitaciones Dobles, Habitaciones Triples, etc.

**¿Para qué sirve?**

- Podrás clasificar tus habitaciones según las características con las que cuenta.
- Una vez establecidas las categorías, podrás asignar una Tarifa para todas las habitaciones que se vendan con esa clasificación.
- Podrás asignar fotos a todas las habitaciones que tengan las mismas características y coincidan en el Tipo de habitación.
- Al presupuestar, podrás ver cuántas habitaciones tienen disponibles para cada Tipo de habitación.
- Ofrecer en el Motor de Reservas categorías de habitaciones.
