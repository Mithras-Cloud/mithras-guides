---
title: "¿Cómo agregar el código de seguimiento de Google Tag Manager a Pxsol?"
description: "Si ya trabajas con Google Tag Manager, en esta guía aprenderás a vincularlo a Pxsol."
---

<Tip>
  **Google Tag Manager** es un sistema gratuito de gestión de etiquetas que te permite actualizar de forma rápida y sencilla códigos de seguimiento en tu sitio web o motor de reservas. [Más información aquí](https://support.google.com/tagmanager/answer/6102821?hl=es&ref_topic=3441530).

Si aún no tienes una cuenta de Google Tag Manager, puedes [crearla aquí](https://marketingplatform.google.com/about/tag-manager/).

</Tip>

<Steps>
  <Step title="Obtener el ID de contenedor de Google Tag Manager">
    Dentro de Google Tag Manager, deberás obtener el ID de tu contenedor.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-40-32-01-PM.png" alt="Obtener ID de contenedor en Google Tag Manager" />
    </Frame>

  </Step>

<Step title="Acceder a Pxsol">Ingresa a [pms.pxsol.com](https://pms.pxsol.com/).</Step>

  <Step title="Navegar a la aplicación 'Puntos de venta'">
    Haz clic en el buscador (ominibox) y luego selecciona la aplicación "Puntos de venta".

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-43-35-91-PM.png" alt="Buscador Pxsol" />
    </Frame>

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-44-44-42-PM.png" alt="Aplicación Puntos de venta Pxsol" />
    </Frame>

  </Step>

  <Step title="Editar el Punto de Venta">
    Abre la edición del Punto de Venta donde deseas colocar el contenedor de Google Tag Manager.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-45-48-54-PM.png" alt="Editar Punto de Venta Pxsol" />
    </Frame>

  </Step>

  <Step title="Ir a la pestaña 'Analítica'">
    Haz clic en la pestaña "Analítica".

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-47-24-12-PM.png" alt="Pestaña Analítica en Pxsol" />
    </Frame>

  </Step>

  <Step title="Colocar el contenedor de Google Tag Manager">
    Pega el ID de contenedor que obtuviste de Google Tag Manager en el campo correspondiente.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-49-22-74-PM.png" alt="Campo Google Tag Manager en Pxsol" />
    </Frame>

  </Step>

  <Step title="Guardar los cambios">
    Haz clic en el botón para guardar los cambios realizados.

    <Frame>
      <img src="https://f.hubspotusercontent40.net/hubfs/7398646/image-png-Jul-20-2021-12-50-14-46-PM.png" alt="Guardar cambios en Pxsol" />
    </Frame>

    <Check>
      ¡Listo! Tu cuenta de Google Tag Manager está vinculada a Pxsol. A partir de ahora, cualquier etiqueta de seguimiento que agregues a GTM estará automáticamente vinculada a Pxsol.
    </Check>

  </Step>
</Steps>

<Warning>
  Si tienes un sitio web y un motor de reservas de Pxsol, este es todo el proceso que debes seguir.
  Si, en cambio, tienes un sitio web de terceros, este proceso solo vinculará tu motor de reservas
  de Pxsol. En ese caso, deberás solicitarle a tu webmaster que vincule GTM a tu sitio web externo.
</Warning>
