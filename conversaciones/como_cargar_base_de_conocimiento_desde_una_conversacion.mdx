---
title: "¿Cómo cargar Base de Conocimiento desde una conversación?"
description: "En esta guía explicaremos cómo adicionar una pregunta y respuesta para darle información al BOT de Conversaciones, desde una conversación con un Huésped."
---


<p><strong>1-&nbsp;</strong>Accede desde el&nbsp;<strong>menú lateral&nbsp;</strong>a la aplicación&nbsp;<strong>Conversaciones</strong></p>
<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jul-26-2023-04-12-33-4255-PM.png?width=371&amp;height=208&amp;name=image-png-Jul-26-2023-04-12-33-4255-PM.png" alt="Acceso a la aplicación Conversaciones" />
</Frame>

<p><strong>2-&nbsp;</strong>Una vez dentro de la aplicación, haz clic en el menú lateral en el apartado&nbsp;<strong>Clientes</strong></p>
<Frame>
<img src="https://ayuda.pxsol.com/hs-fs/hubfs/image-png-Jul-26-2023-04-20-33-2975-PM.png?width=247&amp;height=229&amp;name=image-png-Jul-26-2023-04-20-33-2975-PM.png" alt="Apartado Clientes" />
</Frame>

<Note>
<strong>Importante</strong>, para utilizar la parte de chat con clientes es necesario tener vinculado un número de WhatsApp. Por el momento, la vinculación la realiza el equipo de adquisición. Aparecerá un cartel con el enlace para comunicarse con ventas.
<Frame>
<img src="https://ayuda.pxsol.com/hubfs/image-png-Jul-26-2023-08-42-12-2744-PM.png" alt="Vinculación de número de WhatsApp" />
</Frame>
</Note>

<p><strong>3-&nbsp;</strong>Las nuevas conversaciones entrantes llegarán a la carpeta&nbsp;<strong>Sin asignar</strong></p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-55-47-7233-PM.png" alt="Carpeta Sin asignar" />
</Frame>

<p><strong>4-</strong> Dentro de la conversación, selecciona el <strong>desplegable</strong> de la pregunta o respuesta que deseas cargar en la Base de Conocimiento. Luego, selecciona <strong>Base de Conocimiento</strong>.</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-59-44-0831-PM.png" alt="Seleccionar Base de Conocimiento" />
</Frame>

<p><strong>5-</strong> Se abrirá una ventana lateral donde se cargará la Pregunta y Respuesta seleccionada, corregida por IA. Selecciona la Base de Conocimiento donde se cargará esta información. Corrige las Preguntas y Respuestas a tu gusto.</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-06-59-22-5779-PM.png" alt="Ventana lateral para cargar Pregunta y Respuesta" />
</Frame>

<p><strong>6-</strong> Selecciona <strong>Agregar</strong>.</p>
<Frame>
<img src="https://7398646.fs1.hubspotusercontent-na1.net/hubfs/7398646/image-png-Apr-10-2024-07-01-49-5874-PM.png" alt="Botón Agregar" />
</Frame>

<Note>
La Pregunta y Respuesta se cargará como un nodo de información en la aplicación Base de Conocimiento y servirá para futuras preguntas similares.
</Note>
