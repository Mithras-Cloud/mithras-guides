Esta guía detalla cómo puedes identificar mensajes que no se enviaron correctamente y cómo reintentar su envío de forma manual.

## Reintentar un Mensaje Fallido

Sigue estos pasos para reenviar un mensaje que no se pudo entregar.

<Steps>
  <Step title="Identifica el mensaje fallido">
    Ingresa a la conversación y localiza el mensaje que muestra el estado **fallido** o un indicador de error.
  </Step>

  <Step title="Inicia el reintento">
    Haz clic en la opción **Reintentar envío** que aparece junto al mensaje. El sistema intentará enviar el mensaje de nuevo.

    <Frame caption="Opción 'Reintentar envío' en un mensaje fallido.">
      <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/229906149338e9114da9b91e5832ee84e0866d62a6f.png?X-Amz-Content-Sha256=UNSIGNED-PAYLOAD&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIAQI4Y7PQJFLSSFFFR%2F20260115%2Fus-west-2%2Fs3%2Faws4_request&X-Amz-Date=20260115T202721Z&X-Amz-SignedHeaders=host&X-Amz-Expires=1200&X-Amz-Signature=cf017ab3395f4eaf3fba27008ce45b48e3e057af955f2f13a6fbcdca1a9285ad" alt="Ejemplo de un mensaje fallido con la opción para reintentar el envío." />
    </Frame>
  </Step>

  <Step title="Verifica el resultado">
    <Check>
      Espera la respuesta del sistema. Si el reintento es exitoso, el estado del mensaje cambiará de "fallido" a "entregado" o "leído".
    </Check>
  </Step>
</Steps>

## Límite de Reintentos

<Info>
  El sistema permite reintentar manualmente el envío de un mensaje hasta **3 veces**. Cada vez que seleccionas "Reintentar envío", el sistema contabiliza el intento de forma automática.
</Info>
