## Introducción

En esta guía aprenderás a **agregar y configurar el módulo Distances** en tu sitio web creado con el **Editor Web** de Pxsol. Este módulo te permite mostrar puntos de interés cercanos y personalizar cómo se presentan según tus necesidades.

## Antes de comenzar

Antes de continuar, asegúrate de:

* Tener acceso al módulo **Distances** en tu cuenta de Pxsol.
* Contar con permisos de edición en la página donde deseas agregar el módulo.

## ¿Dónde se encuentra esta función?

Puedes acceder a esta funcionalidad desde:

**Editor Web → Panel de módulos → Distances**

## Cómo acceder al módulo

Para empezar a utilizar el módulo Distances, primero debes agregarlo a tu página.

### 1. Ingresa al editor de tu sitio

Navega a la página donde deseas agregar el módulo y abre el editor de contenido.

## Configuración del módulo

Una vez que hayas agregado el módulo, puedes personalizar su comportamiento y apariencia desde el panel de configuración.

### Opciones de visualización

Ajusta cómo se presentan los datos para ofrecer la mejor experiencia a tus visitantes.

* **Mostrar u ocultar íconos:** Activa o desactiva la visualización de íconos junto a cada punto de interés.
* **Ordenar por distancia:** Organiza automáticamente la lista para mostrar primero los lugares más cercanos.
* **Limitar número máximo de elementos:** Define cuántos puntos de interés se mostrarán como máximo para evitar sobrecargar la página.

<Frame>
  <img src="https://files-public-web.s3.us-west-2.amazonaws.com/1978/company/library/user/886719056514ecef8c3e6ff169db761eebe3077aef.png" alt="Imagen" />

### Diseño

Selecciona el formato que mejor se adapte al estilo de tu sitio web. Puedes elegir entre:

* **Lista:** Un formato vertical clásico, ideal para mostrar detalles.
* **Grilla:** Organiza los elementos en una cuadrícula, perfecto para un diseño visualmente equilibrado.
* **Compacto:** Una vista minimizada para ahorrar espacio.

Adicionalmente, puedes configurar los siguientes aspectos visuales:

* Color del ícono
* Color de fondo del elemento
* Radio de los bordes para un estilo redondeado o cuadrado

<Tip title="Recomendaciones de uso">
  Para asegurar una funcionalidad y diseño óptimos, considera lo siguiente:

  * **Ordena por distancia:** Activa siempre esta opción para que la información sea más relevante para el usuario.
  * **Limita los resultados:** No muestres demasiados puntos. Un número entre 5 y 10 suele ser suficiente.
  * **Mantén un diseño limpio:** Asegúrate de que los colores y el formato elegidos sean coherentes con el resto de tu sitio web.
</Tip>

## Publica tus cambios

Cuando termines de configurar el módulo, no olvides guardar tu trabajo.

Haz clic en el botón **Publicar página** ubicado en la esquina superior para que los cambios sean visibles en tu sitio web.
