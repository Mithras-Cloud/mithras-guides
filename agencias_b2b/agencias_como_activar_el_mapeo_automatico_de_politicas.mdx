---
title: "Agencias: Cómo activar el mapeo automático de políticas"
description: "En esta guía se explica cómo habilitar el mapeo automático de políticas y, de forma predeterminada, el enmascaramiento de tarifarios."
---

El **mapeo automático** de políticas adapta dinámicamente las políticas de los hoteles en tiempo real durante cada búsqueda. Considera las variables de configuración de cada hotel para asignarles una política equivalente o similar de la agencia, garantizando el cumplimiento de las restricciones originales. Esto asegura que las políticas mostradas en la agencia reflejen con precisión las configuraciones proporcionadas por cada hotel. Además, se implementa el enmascaramiento de tarifas.

Para activar el mapeo automático, accede a la aplicación "Mapeo de políticas". En la parte superior derecha, encontrarás el interruptor para habilitar esta función.

- Mapeo automático desactivado

Una vez activada, al realizar una búsqueda en el motor de reserva o en la aplicación presupuestar de la agencia, podrás visualizar correctamente la aplicación del mapeo automático junto con el enmascaramiento de tarifas.

<Warning>
  ¡Importante! Si aplicas un mapeo manual, este anulará el mapeo automático asociado, lo que
  significa que las configuraciones manuales prevalecerán sobre las automáticas.
</Warning>
