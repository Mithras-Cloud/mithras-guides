<p>&nbsp;</p>

<p>El web check in permite realizar el tradicional proceso del Check In, de manera online.</p>
<p>De manera sencilla, tu huésped podrá completar la ficha de registro antes del arribo, en la Web App del hotel.</p>
<p>Ingresará con número de reserva y apellido o mediante link para iniciar el proceso.</p>
<p>&nbsp;</p>
<p>A través del proceso de Web Check In vamos a poder:</p>
<ul>
  <li>Escanear los documentos del huésped.</li>
  <li>Autocompletar los datos del huésped.</li>
  <li>El huésped podrá firmar digitalmente los términos y condiciones de estadía en la propiedad.</li>
</ul>
<p>&nbsp;</p>
<p>Ventajas de tener el proceso de Check In de manera Online:</p>
<ul>
  <li>Disminuye el tiempo de contacto entre los huéspedes con los recepcionistas del hotel, permitiendo el cumplimiento de los protocolos establecidos para la reapertura de los hoteles que sugieren el uso de estas herramientas.</li>
  <li>Aumenta la eficiencia de nuestro equipo, reduciendo el tiempo que dedica nuestro personal a realizar este proceso.</li>
  <li>Mejora la experiencia del huésped, permitiendo disfrutar de las instalaciones del hotel desde el momento que llega.</li>
  <li>Permite la detección temprana de reservas que podrían realizar No Show.</li>
</ul>

<p><strong>Email al huésped.</strong></p>
<p>El enlace para realizar el Check In online se podrá enviar de dos formas:</p>
<ul>
  <li>El sistema enviará un correo al huésped automáticamente, 48 hs antes del horario de ingreso al hotel, con el enlace para que realice su Check In online. El correo que se enviará al cliente tendrá la siguiente información:
    <ul>
      <li>Nombre del hotel.</li>
      <li>Nombre del huésped.</li>
      <li>Fecha de Ingreso.</li>
      <li>Fecha de egreso.</li>
      <li>Duración.</li>
      <li>Dirección del hotel.</li>
      <li>Nº de Teléfono del hotel.</li>
      <li>Nº de Reserva.</li>
      <li>Enlace para realizar el proceso de Check In.</li>
    </ul>
  </li>
</ul>

<p><strong>Notificación al hotel.</strong></p>
<p>Una vez realizado el Check In online, el hotel recibirá un aviso de dos maneras:</p>
<ul>
  <li>Mediante un correo electrónico dirigido a los usuarios del hotel que estén configurados para recibir esas notificaciones.</li>
  <li>En el historial de comunicación con el cliente en el CRM. Podrán observar en la pestaña Historial de la reserva la confirmación del Check In realizado y en la pestaña Huéspedes toda la información cargada por el huésped.</li>
</ul>
<p>&nbsp;</p>
<p><strong>Arribo al hotel.</strong></p>
<p>Una vez recibida la notificación del Check In realizado por el cliente, solamente queda esperar el arribo del mismo al hotel.</p>
